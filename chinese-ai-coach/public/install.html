スマホにアプリを追加する

台湾スピークはWebアプリです。
ホーム画面に追加すると、ネイティブアプリのように使えます。

iPhone（Safari）の場合

  1. Safariで taiwanspeak.ryugaku101.com を開く
  2. 画面下の 共有ボタン（四角に上矢印のアイコン）をタップ
  3. メニューを下にスクロールして 「ホーム画面に追加」 をタップ
  4. 名前を確認して 「追加」 をタップ
  5. ホーム画面に台湾スピークのアイコンが追加されます
Safari以外のブラウザ（Chrome、LINE内ブラウザ等）からは「ホーム画面に追加」が表示されません。必ずSafariで開いてください。

Android（Chrome）の場合

  1. Chromeで taiwanspeak.ryugaku101.com を開く
  2. 画面上部に 「アプリをインストール」 バナーが表示されたらタップ
  3. バナーが出ない場合: 右上の メニュー（3点） → 「ホーム画面に追加」 をタップ
  4. 「インストール」 をタップして完了

PC（Chrome / Edge）の場合

  1. ブラウザで taiwanspeak.ryugaku101.com を開く
  2. アドレスバー右側の インストールアイコン をクリック
  3. 「インストール」 をクリックして完了
