  3-7-14ルールとは
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  面談後、以下の4タイミングで必ず連絡する仕組み。

  ┌────────┬──────────────────────────────────┐
  │ タイミング │ 目的                                          │
  ├────────┼──────────────────────────────────┤
  │ 当日中     │ お礼＋面談要約＋次のアクション明示            │
  │ 3日後      │ 家族相談の進捗確認＋追加質問の受付            │
  │ 7日後      │ 枠の期限通知＋障壁の特定                     │
  │ 14日後     │ 最終確認＋次回説明会への誘導                  │
  └────────┴──────────────────────────────────┘

  ★ 鉄則：
    ・相手が返信しなくても、次のタイミングで必ず送る
    ・「お待ちしております」で終わらない
    ・毎回「次のアクション」を必ず提示する
    ・連絡しない＝「あなたのことは忘れました」と同じ

  ★ これが最も重要なセクション。
    旧スタイルの最大の問題は、キャンセルを即受理していたこと。

  ──────────────────────────────────────
  ✕ 絶対にやってはいけない返し方
  ──────────────────────────────────────

    ✕ 「承知いたしました。ご連絡ありがとうございます。
       またの機会にお待ちしております。」

    → これは「もう二度と連絡してくるな」と同義。
      相手は「やっぱりやめておこう」の気持ちが確定する。

    ✕ 実際にあった最悪の事例（佐藤瑞季ケース）：
      相手「私には通い切れる自信がないので、この件は無しにしてください」
      リッキー「承知いたしました。ご連絡ありがとうございます。
              またの機会にお待ちしております」
      → 即日キャンセル確定。再検討の余地ゼロ。

  ──────────────────────────────────────
  ✅ 正しい返し方（3ステップ）
  ──────────────────────────────────────

  【ステップ1：受容 → 共感 → 理由の質問】

    〇〇さん

    ご連絡ありがとうございます。
    お気持ちは十分理解できます。

    大きな決断なので、慎重になるのは当然ですし、
    〇〇さんが納得できる形で進むことが一番大事です。

    一つだけお願いがあるんですが、
    差し支えなければ、今回決めきれなかった一番の理由を
    教えていただけませんか？

    無理にお引き止めするつもりは全くないです。
    今後のサービス改善の参考に、ぜひ聞かせてください。

  【ステップ2：理由に応じた再提案】

    ┌─────────────────────────────────────────┐
    │ キャンセル理由別の再提案                              │
    ├─────────────────────────────────────────┤
    │                                                       │
    │ 「お金が厳しい」                                      │
    │  → 「実は分割払いの柔軟な対応も可能です。             │
    │     例えば頭金20万円で始められるケースもあります。     │
    │     一度、お支払いプランだけでもご相談しませんか？」   │
    │                                                       │
    │ 「通い切れる自信がない」（佐藤瑞季パターン）          │
    │  → 「その不安、すごくわかります。                     │
    │     実は同じ不安を持っていた子が今台湾で              │
    │     楽しく過ごしています。                             │
    │     よければ、その子の体験談をお送りしてもいいですか？ │
    │     読んでみて、もし気持ちが変わらなければ             │
    │     それで全然OKです。」                               │
    │                                                       │
    │ 「家族に反対された」                                  │
    │  → 「そうでしたか。どなたがどんな理由で               │
    │     反対されていますか？                               │
    │     保護者の方向けの資料や、                           │
    │     直接お話しする機会もご用意できます。               │
    │     反対の理由が情報不足から来ているなら、             │
    │     解消できるかもしれません。」                       │
    │                                                       │
    │ 「他のエージェントにした」                             │
    │  → 「なるほど、ご判断されたんですね。                 │
    │     もし今後、そちらのサービスで                       │
    │     何か気になることが出てきたら、                     │
    │     いつでもご相談ください。                           │
    │     台湾留学のことなら何でもお答えします。」           │
    │                                                       │
    │ 「今はタイミングじゃない」                             │
    │  → 「わかりました。ちなみに、                         │
    │     来年の6期生の募集は〇月頃を予定しています。       │
    │     その時期にご案内をお送りしてもいいですか？」       │
    │                                                       │
    └─────────────────────────────────────────┘

  【ステップ3：ドアを開けておく】

    どの理由の場合でも、最後に必ずこう締める：

    「〇〇さんの留学への気持ちは本物だと感じました。
     状況が変わったり、もう一度話を聞きたいと思った時は、
     いつでもこのLINEに連絡してくださいね。
     いつでもお待ちしています。」

    ★「またの機会に」と「いつでもお待ちしています」の違い：
    前者＝社交辞令。相手は二度と連絡しない。
    後者＝具体的な行動の提案。「連絡していいんだ」と思える。

  フォローLINEの基本ルール
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ──────────────────────────────────────
  ✅ やるべきこと
  ──────────────────────────────────────

  1. 毎回「面談で話した具体的な内容」に触れる
     → 「先日お話しした△△のこと」と書くだけで
       「ちゃんと覚えてくれてる」と感じる

  2. 毎回「次のアクション」を提示する
     → 「〇日までにお返事を」「質問があれば送ってください」

  3. 質問は具体的に聞く
     ✕「何かありますか？」
     ✅「費用面ですか？お子さんの気持ちですか？他社との比較ですか？」

  4. 相手の名前を使う
     → 「〇〇さん」「夏紀ちゃん」など。定型文感を消す。

  5. 相手の状況に合わせた情報を送る
     → 成績が低い子には合格実績
     → 費用が心配な子には奨学金情報
     → 一人っ子なら寮生活の安心感

  ──────────────────────────────────────
  ✕ やってはいけないこと
  ──────────────────────────────────────

  1. 「お待ちしております」で終わる
     → 待つな。能動的に動け。

  2. 定型文をそのまま送る
     → 福井ケースの「ご不明点ございましたら、お気軽に
       お申しつけください！」は定型文丸出しで響かない。

  3. 長文を一括送信する
     → 短いメッセージを複数回に分けて送る方が読まれる。

  4. キャンセルを即受理する
     → 必ず理由を聞く。理由を聞くだけで30%は再検討する。

  5. 「お忙しいところすみません」を多用する
     → 1回目はOK。2回目以降は「すみません」より
       「〇〇さんのことが気になっていたので」の方がいい。

  6. 相手が返信しないからと連絡をやめる
     → 3日後・7日後・14日後は返信がなくても必ず送る。
       ただし14日後で反応がなければ、月1回の情報提供に切り替える。

  月次フォロー（14日後以降）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  14日後のフォローで成約に至らなかった場合、
  月1回の情報提供で関係を維持する。

  ──────────────────────────────────────
  テンプレート M：月次情報提供
  ──────────────────────────────────────

    〇〇さん、こんにちは！
    101センターのリッキーです。

    〇月のニュースをお届けしますね。

    📌 [近況トピック1つ]
    例：「5期生の△△さんが台湾で初めてのプレゼンを成功させました！」
    例：「サマーキャンプの日程が確定しました（7/26〜8/19）」
    例：「△△大学の奨学金情報が更新されました」

    もし台湾留学のことで
    また話を聞きたいなと思ったら、
    いつでもこのLINEに連絡してくださいね。

  ★ 月次フォローの目的：
    ・「忘れられていない」と思わせる
    ・新しい情報で再検討のきっかけを作る
    ・将来の紹介（友人・兄弟）につなげる
