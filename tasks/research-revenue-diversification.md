## 1. 留学ローン × 金融機関提携

### 1-1. 法的に要注意（会議で未議論の重要論点）

**銀行から紹介手数料を受け取ると「銀行代理業」の許可が必要になるリスクがある。**

金融庁監督指針（VIII-3-2-1-1(3)）により、銀行から直接・間接の報酬を受けて顧客を紹介する行為は、形式的に紹介であっても銀行代理業に該当しうる。

### 1-2. 現実的な3ルート

| ルート | 概要 | ハードル |
|---|---|---|
| **① 無報酬の紹介（Phase 1推奨）** | パンフ配布・リンク案内のみ。手数料ゼロだが法的リスクもゼロ。留学サービスの成約率を上げる武器に | 低 |
| **② 信販会社の加盟店契約** | オリコ（2,200校提携）・ジャックス（700校提携）。学校法人前提のため株式会社が入れるか要確認 | 中 |
| **③ 取引先信金との業務提携** | 手数料ではなく「金利優遇プラン」を顧客に提供。**芝信用金庫（既存取引先）**・肥後銀行が候補 | 中 |

#### 教育ローン商品（2種類）

**① 教育カードローン「まなぶ」**
- 融資限度額: 50万〜500万円
- 金利: 変動金利
- カードローン期間: 最長5年（6年制学部は7年）
- 用途: 子弟等の教育資金
- 担保・保証人: 不要

**② 生活設計ローン（教育ローン枠）**
- 融資限度額: 最高1,000万円
- 金利: 年2.75%（3年以内）/ 3.25%（5年以内）/ 3.75%（16年以内）
- 用途: 学校納付金、就学付帯費用
- 担保・保証人: 不要（しんきん保証基金）

**留学向け利用**: 明示的な記載なし。「教育に関する資金」と広く記載 → 窓口確認推奨

#### 提携の強み（既存取引先だからこそ）
- **新規飛び込みではなく「既存顧客からの相談」** → ハードルが圧倒的に低い
- 法人取引の実績があるため、信用情報の蓄積がある
- 芝信金は「しばしんビジネスサポートネット」で取引先のPR・販路拡大支援を提供
- 商談会・ビジネス交流会の開催実績あり → 提携提案の場として活用可能
- JETROとの連携で海外展開支援も行っている → 台湾留学との親和性

### 1-5. 熊本 × TSMC — 肥後銀行の方が筋が良い

- **肥後銀行**: 2022年に台湾・玉山銀行と業務提携。2023年に台北事務所開設
- **熊本信用金庫**: 台湾関連の動きは確認できず。ただしTSMC所在の菊陽町が営業エリア内
- **熊本県**: 2025年2月に玉山フィナンシャルHDと包括連携協定

### 1-6. 注目事例 — 埼玉県モデル

埼玉県が自治体として3金融機関（埼玉りそな・武蔵野銀行・埼玉縣信金）と留学ローン提携を実現。県が「対象者確認書」を発行→金利優遇（年2.30%〜2.55%）。民間がゲートキーパー機能を担えれば転用可能。

### 1-7. 推奨アクション

**Phase 1（すぐ）**: 日本政策金融公庫の教育ローン案内を全顧客に標準実施（無料・法的リスクなし）
**Phase 2（3ヶ月内）**: **芝信用金庫の担当者に相談**（既存取引先の強みを活かす）/ オリコ・ジャックスに加盟店契約を打診
**Phase 3（半年〜）**: 肥後銀行にTSMC文脈で台湾留学ローン提携を提案

## 2. 補習・個別指導サービス（最大の発見）

### 2-1. 市場の空白

**「台湾の大学の授業についていくための補習」に特化したサービスは事実上存在しない。**

- 海外子女向けオンライン家庭教師（EDUBAL、トライ等）は存在するが、全て「帰国子女の日本の受験対策」が主目的
- 台湾留学エージェントのアフターサポートは「生活支援」止まりで、学習支援まで踏み込んでいるケースは極めて少ない
- 留学エージェントの月額サブスクモデルも検索範囲では前例なし

### 2-2. 価格設定

| プラン | 内容 | 月額 |
|---|---|---|
| **ライト** | 週1回グループ指導 | 9,800〜15,000円 |
| **スタンダード** | 週2回少人数指導 | 19,800〜25,000円 |
| **プレミアム** | 週2回＋個別月2回 | 29,800〜39,800円 |

根拠: オンライン家庭教師の相場（月5,000〜40,000円）、個別指導塾の中高生相場と一致

### 2-3. 収益シミュレーション（50名ベース）

| シナリオ | 月額 | 加入率 | 人数 | 月商 | 年商 |
|---|---|---|---|---|---|
| 控えめ | 15,000円 | 50% | 25名 | 375,000円 | **450万円** |
| 標準 | 19,800円 | 60% | 30名 | 594,000円 | **713万円** |
| 強気 | 25,000円 | 70% | 35名 | 875,000円 | **1,050万円** |

### 2-4. コスト構造（30名加入時の概算）

| 項目 | 月額 |
|---|---|
| 売上 | 594,000円 |
| 講師人件費（3名、時給2,000円×月40h） | △240,000円 |
| プラットフォーム・通信費 | △20,000円 |
| 運営管理（教材作成等） | △50,000円 |
| **営業利益** | **約284,000円（利益率48%）** |

### 2-5. 親が払える根拠

- 台湾留学の年間費用は約100〜150万円
- 日本の塾相場（月1.5〜4万円）を親は経験済み
- 「子どもが授業についていけない」は切実 → 月2万円は留学費用の15〜20%追加で受容範囲
- 会議でショウさんも「50万出してるならあと10万は必要経費」と発言

### 2-6. 実現のポイント

- **多学科対応が課題**: 国際系、ビジネス系、工学系など学科が多岐にわたる
- **現地人材の確保**: 台湾在住のバイリンガル講師（日本語＋中国語）が必要
- **学生寮の空き部屋を活用**: 常設の「学習支援拠点」として運用（会議で議論済み）
- **学習塾業界を研究**: 課外授業の工夫を他業界から学ぶ（会議でのアクションアイテム）

## 3. 他業界オプションサービス

### 3-1. 卒業生の就職紹介（成功報酬型）

| 項目 | 内容 |
|---|---|
| 紹介手数料の相場 | 理論年収の **30〜35%**（年収400万→手数料120〜140万円） |
| 必要な免許 | 有料職業紹介事業許可（申請〜許可2〜3ヶ月） |
| 資金要件 | 純資産500万円以上、現金150万円以上 |
| 台湾・中国語人材の需要 | グローバルリーフ、えみたす等が既に市場を形成 |
| ポイント | 留学→就職まで一気通貫サポートは大きな差別化 |

### 3-2. 保護者向け月額見守りサービス

| 項目 | 内容 |
|---|---|
| 参考モデル | OSSMA（大学向け危機管理サービス）月額3,630円〜 |
| 内容 | 月次レポート＋緊急時24時間対応＋安否確認 |
| 価格帯 | 月額3,000〜5,000円 |
| 年間収益 | 50世帯で月15〜25万円 → 年180〜300万円 |
| 追加案 | 保護者コミュニティ（年額制）、台湾親子旅行企画 |

### 3-3. 留学保険の代理店販売

- 5年間の在学期間中、毎年更新でマージンが継続発生
- 旅行業界のオプション収益モデルと同構造
- SIMカード手配（5,500円/件）、空港送迎（44,000円/件）もバンドル可能

### 3-4. OB/OGコミュニティ

- エディクム（留学エージェント）がAlumni Associationを運営。就職セミナー・異業種交流会を展開
- 年額制で月5〜10万円のストック収益が見込める

### 3-5. 競合の料金参考（毎日留学ナビ・台湾）

| サービス | 料金 |
|---|---|
| 入学申請代行 | 44,000円 |
| 到着サポート | 44,000円 |
| 生活フルサポート12ヶ月 | 121,000円 |
| SIMカード手配 | 5,500円 |
| 営業時間外対応 | 6,600円/回 |

### 3-6. 留学エージェント業界の経営数値

- 月間平均利益75万円、年間営業利益約1,000万円、売上2,500〜5,000万円が中堅規模
- 世界市場: 2023年168億ドル → 2030年272億ドル（CAGR 7.36%）

## 5. 推奨ロードマップ

### Phase 1: すぐできること（〜1ヶ月）
- [ ] 日本政策金融公庫の教育ローン案内を全顧客に標準実施
- [ ] 学生アンケート実施（ローン利用率・補習ニーズの定量把握）
- [ ] 補習サービスの概念設計（プラン・価格・講師要件）

### Phase 2: 基盤構築（1〜3ヶ月）
- [ ] 補習サービスのパイロット版を5期生で試行
- [ ] 千葉信用金庫に業務提携を提案
- [ ] オリコ・ジャックスに加盟店契約を打診
- [ ] 保護者向け見守りサービスの設計

### Phase 3: 拡大（3〜6ヶ月）
- [ ] 補習サービスの本格展開（価格・講師体制の確定）
- [ ] 有料職業紹介事業の免許取得準備
- [ ] 肥後銀行にTSMC文脈で台湾留学ローン提携を提案

### Phase 4: 収穫（6ヶ月〜）
- [ ] 就職紹介サービス開始
- [ ] OB/OGコミュニティ（年額制）立ち上げ
- [ ] 全オプションのバンドル販売開始
