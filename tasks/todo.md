## 現状サマリー

- **4期生実績**: AIO25名(¥3,725万) + サマキャン(¥1,060万) + 春キャン(¥220万) ≒ ¥5,000万
- **純利益率**: 2.9%（業界平均10-15%。構造的に危険）
- **固定費比率**: 44.4%（健全ライン25-35%超過）
- **広告予算**: 月¥100万
- **目標**: 年間売上¥1億

### おさる式 売上の方程式

```
売上 = リスト数 × CVR（成約率） × LTV
```

| 指標 | 現在 | 目標 |
|------|------|------|
| LP登録率 | 0.95%（OW直） / 6.74%（ガイドブック） | ガイドブック経由に一本化 |
| 面談成約率 | 42% (5/12) | 55-60% |
| AIO成約数 | 25名/年 | 35-40名/年 |
| メール開封率 | 38-60%（非常に高い） | CTR改善が課題（現状1%前後） |
| LINE配信枠利用率 | 0.58% | 20%+ |

### [ ] 2-4. YouTube対談動画 第1本の撮影・公開
