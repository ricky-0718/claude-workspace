# YouTube 対談動画 収録・編集チェックリスト

台湾留学生のインタビュー動画（対談動画）を効率的に制作・公開するための統合チェックリスト。

---

## Phase 1: 収録前チェックリスト

### 共通（Zoom/対面）

- [ ] 撮影・公開同意書の取得（生徒＋保護者。未成年は保護者署名必須）
- [ ] 事前ヒアリングシートの回収確認（youtube-taidan-hearing-sheet.md を使用）
- [ ] ヒアリングシートの内容を確認し、深掘りポイントをメモ
- [ ] 合格通知・TOCFL証明書等のテロップ用素材を事前収集
- [ ] 出願校と合否結果の一覧データ確認
- [ ] LINE登録URL・QRコード画像の準備
- [ ] 特典コンテンツの存在確認（5個）
- [ ] 台本テンプレート（youtube-taidan-template.md）に生徒データを記入

### Zoom版追加

- [ ] 通信環境の事前テスト（生徒・保護者とも）
- [ ] 録画設定確認（ギャラリービュー/スピーカービュー）
- [ ] バーチャル背景 or 部屋の背景確認
- [ ] 音声テスト（イヤホン推奨）

### 対面版追加

- [ ] ロケーション選定（大学キャンパス、カフェ、寮等）
- [ ] カメラ・マイク機材準備
- [ ] 背景に映る台湾らしい要素の確認
- [ ] 予備バッテリー・SDカード

---

## Phase 2: 収録当日チェックリスト

- [ ] 機材セットアップ・テスト撮影
- [ ] 生徒・保護者にリラックスしてもらう（アイスブレイク5分）
- [ ] 「きれいに話さなくて大丈夫、編集でカットできます」と伝える
- [ ] 録画開始
- [ ] 12ステップに沿って進行
- [ ] 親パートへの切り替えタイミングを確認
- [ ] テロップ候補のメモ取り（印象的な発言、数字）
- [ ] 対面版: 中国語で一言話してもらう（演出用）
- [ ] 録画停止・データ保存確認

---

## Phase 3: 編集・公開チェックリスト

### 編集

- [ ] テロップ追加（重要キーワード・数字は必ず）
- [ ] BGM設定（爽やか系。親パートは落ち着いたトーン）
- [ ] 余分な間・言い直し・噛みをカット
- [ ] TOCFL証明書・合格通知の画面表示
- [ ] 出願校一覧テロップの作成・表示
- [ ] LINE QRコード: CTA時に大きく、それ以外は右下小
- [ ] 101センターロゴ: 冒頭＋右下常時表示
- [ ] 合計尺が15-18分に収まっているか確認

### 公開準備

- [ ] サムネイル作成（3パターンから選定。13-15文字以内）
- [ ] タイトル決定
- [ ] 概要欄記入（youtube-description-template.md を使用）
- [ ] タイムスタンプを実測値で記入
- [ ] LINE登録URLが概要欄冒頭3行に入っているか確認
- [ ] ハッシュタグ設定
- [ ] 公開設定（公開/限定公開）

### 公開後

- [ ] SNS横展開（Instagram、X、TikTok）
- [ ] LINE配信で告知
- [ ] コメント対応（最初の24時間は特に注視）
- [ ] 視聴データ確認（3日後、7日後、30日後）
