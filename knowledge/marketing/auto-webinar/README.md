## ファネル構造

```
Meta広告 → オプトインLP → LINE登録 → LINEステップ配信 → 日程選択 → ウェビナー視聴 → 個別相談申込 → 面談 → 成約
```

### 核心的な診断

> **広告は正しく動いている。問題は広告の「後」にある。**
> LP→LINE登録の変換率（0.91%）が最大のボトルネック。
> ファネル後半（説明会〜成約）は極めて優秀。
