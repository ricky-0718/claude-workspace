**台湾留学 大学診断カルテ**
**— あなたの条件で狙える大学がわかる —**

台湾には100以上の大学がある。
でも、あなたに合う大学は、その中のほんの数校です。

この診断カルテに自分の条件を書き込むだけで、
**今の成績・語学力・興味から、現実的に狙える大学**が見えてきます。

- 何が足りていて、何が足りないか
- どの大学が現実的で、どこが挑戦校か
- 奨学金のチャンスがあるかどうか

5分で書けて、一生の進路判断に使える。
まずは、自分の「今」を整理するところから始めましょう。

## 2ページ目｜はじめに

こんにちは、リッキーです。

台湾留学を考え始めると、最初にぶつかるのがこの壁です。

**「大学が多すぎて、どこを選べばいいかわからない」**

台湾には100以上の大学があって、学科は何千もある。ネットで調べても情報が多すぎて、むしろ混乱する。親に聞いても台湾の大学のことはわからない。

結果、「なんとなく有名そうな大学」「台北にある大学」「友達が行くと言っている大学」——こういう基準で決めてしまう人がかなり多いです。

でもそれ、めちゃくちゃ危ないです。

大学選びで大事なのは、**「行きたい場所」ではなく「自分の条件で合格できて、4年間続けられる場所」**です。

この診断カルテは、あなたの条件を整理して、データに基づいて「狙える大学」を見つけるためのツールです。自己診断 → マッチング → 行動計画の3ステップで、大学選びの全体像がクリアになります。

**注意：この診断カルテは、あくまで目安です。** 実際の出願設計は、最新の募集要項と個別の状況を踏まえて行う必要があります。でも、「自分がどこを狙えそうか」の見当をつけるには十分な精度があります。

### 大学ごとに求められる残高証明

出願時に「家庭にこれだけの資金があります」と証明する書類が必要です。金額は大学によって違います。

| 大学名 | 必要金額（日本円目安） |
|---|---|
| 雲林科技大学 | 約50万円 |
| 淡江大學 | 約50万円 |
| 中國文化大學 | 約50万円 |
| 東吳大學 | 約60万円 |
| 實踐大學 | 約60万円 |
| 中原大学 | 約62万円 |
| 師範大學 | 約75万円 |
| 中國醫藥大學 | 約78万円 |
| 政治大學 | 約100万円 |
| 輔仁大學 | 約100万円 |

**補足：** これは「その金額を使う」という意味ではなく、「その金額が口座にありますよ」という証明です。親の名義でも三親等以内なら使えます。

保護者の方は、志望大学が決まる前でも、**100万円の残高証明が出せる準備**をしておくと安心です。これがあれば、どの大学にも対応できます。

## 7ページ目｜よくある「思い込み」を壊す

### 思い込み vs データ

大学選びで一番怖いのは、**間違った思い込みで可能性を狭めてしまうこと**です。

| よくある思い込み | データが示す現実 |
|---|---|
| 「評定3.0じゃ国立は無理」 | 雲林科技大学は評定3.0で合格＋奨学金 |
| 「中国語ゼロじゃ出願できない」 | 全英語課程ならTOCFLなしで出願可能 |
| 「奨学金はトップ成績の人だけ」 | 雲林科技大学は合格者全員に奨学金 |
| 「国立大学=超難関」 | 師範大は3期生全員合格（評定3.3〜4.9） |
| 「評定が高ければどこでも受かる」 | 評定4.9でも台湾大は不合格 |
| 「有名な大学に行けば安心」 | 大学名より学科の適性と卒業後が大事 |

### 大事なのは「正しい情報」で判断すること

思い込みで可能性を狭めるのはもったいない。でも逆に、根拠のない楽観で動くのも危ない。**データに基づいて、自分の条件に合った大学を選ぶ。** これがこの診断カルテの目的です。

## 8ページ目｜保護者の方へ

### お子さんと一緒に、このカルテを使ってみてください

保護者の方にとって、台湾留学の大学選びは不安が大きいと思います。日本の大学なら偏差値で比較できるけど、台湾の大学にはそれがない。ネットで調べても中国語の情報ばかり。

だからこそ、この診断カルテを使って、**お子さんと一緒に「今の条件」と「狙える大学」を可視化してみてください。**

### 保護者が確認すべき3つのポイント

**1. 費用の現実**
- 財力証明に必要な金額を確認しましたか？
- 奨学金の可能性がある大学はどこですか？
- 4年間の総費用を試算しましたか？

**2. 安全と環境**
- その大学に日本人学生はいますか？
- 寮はありますか？
- 緊急時のサポート体制はどうなっていますか？

**3. 卒業後の進路**
- その学科を卒業した先に、どんなキャリアがありますか？
- 就職サポートはありますか？
- 台湾に残るのか、日本に帰るのか、話し合いましたか？

**大学選びは、お子さんだけで完結するものではありません。** 保護者の方が費用と安全を把握し、お子さんが学びたい内容と将来を考える。この両輪が揃って初めて、後悔のない大学選びができます。

## 9ページ目｜最後に

### この診断カルテは「スタート地点」です

ここまで書き込んでみて、いかがでしたか？

「自分が狙える大学が見えてきた」と感じた人もいれば、「まだわからないことだらけ」と感じた人もいると思います。

どちらでも大丈夫です。

この診断カルテは、**最終的な出願先を決めるためのものではありません。** 自分の「今」を整理して、「どの方向に進めばいいか」のヒントを見つけるためのツールです。

実際の出願設計は、もっと細かい情報が必要です。

- 今年の最新の募集要項
- 大学ごとの出願時期と書類要件
- あなたの評定とTOCFLの組み合わせで、本当に現実的な大学はどこか
- 奨学金を狙うなら、どういう戦略で出すか

これらは、一人で調べるのは正直かなり大変です。

**だからこそ、この診断カルテを持って説明会に来てください。**

あなたが書き込んだ条件をベースに、最新のデータで出願設計を一緒に組みます。「自分にはどの大学が合っているのか」が、もっとクリアに見えるようになります。

5分で書けるカルテが、あなたの4年間を変えるかもしれません。
