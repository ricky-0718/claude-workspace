## 詳細診断

### 1. メッセージマッチ（55/100）🔴

**問題**: 広告→LPの期待値のギャップ

広告でクリックしたユーザーは「台湾留学」に興味を持ってLPに来る。
しかしLPのファーストビューは**質問形（「このまま日本の大学でいいの？」）**で始まる。

**診断:**
- 広告のCTR 4.94%（優秀）→ 広告の訴求は刺さっている
- LP到達後の離脱が激しい → LPが広告の期待に応えていない
- ファーストビューで「台湾留学の具体的メリット」が直接見えない
- 質問形ヘッドラインは認知層向け。広告クリック者はすでに興味があるので**解決策を見せるべき**

**改善案:**
```
Before: 「このまま日本の大学でいいの？」と悩む高校生へ。
After:  台湾トップ大学に「学費0円」で進学する方法を、無料説明会で公開中。
```
→ 広告クリック者は「知りたい」段階。質問ではなく**答え**を提示する。

---

### 2. ファーストビュー（50/100 → Clarity実測で裏付け済み）🔴

**問題**: ヒーローセクションが2つ存在 + CTA到達が遅い

**Clarity実測データ:**
- **80%がファーストビューで離脱** → ファーストビューが「勝負の場」
- アクティブ滞在時間28秒 → ファーストビューに28秒で判断が完了している
- ほぼ100%がモバイルユーザー → モバイルのファーストビューが最重要

**診断:**
- デスクトップ版ヒーロー（ref_1〜ref_11）とモバイル版ヒーロー（ref_12〜ref_36）が**両方表示されている可能性**
- デスクトップ版ヒーローにはCTAボタンがない（リンクのみ）
- モバイル版ヒーローに初めてCTA「LINEを追加して席を確保する」が登場
- 日程情報がファーストビュー内にあるのは良い
- **ユーザーの99%がモバイルなのに、デスクトップ版ヒーローが先に表示されている**

**改善案:**
- ヒーローセクションを1つに統合（モバイルファースト）
- ファーストビュー内に必ずCTAボタンを配置
- ヘッドライン→サブヘッド→CTA の3要素をスクロールなしで見せる
- **80%がここで判断する → この1画面に全力を注ぐ**

---

### 3. CTA設計（65/100）🟡

**問題**: CTAの文言が5種類ありバラバラ

| 出現箇所 | CTA文言 |
|---|---|
| ヒーロー | （リンクのみ、ボタンなし） |
| 費用比較後 | 「無料で席を確保する」 |
| 実績後 | 「LINEを追加して席を確保する」 |
| 保護者後 | 「LINEを追加して席を確保する」 |
| 説明会内容後 | 「無料で情報を受け取る」 |
| 8大特典後 | 「特典付きで席を確保する」 |
| フローティングバー | 「LINEを追加して席を確保する」 |

**診断:**
- CTAが5種類あると、ユーザーは「何が起こるか」が不明確になる
- 「情報を受け取る」と「席を確保する」は異なる期待値を生む
- 「LINE」という単語がCTAに入ると、LINEアプリのインストールが必要と思い離脱するユーザーも

**改善案:**
- CTA文言を1つに統一: **「無料で席を確保する（LINE）」**
- または行動のハードルが低い: **「30秒で申し込む」**
- CTAボタンの色・サイズを統一し、視認性を最大化

### 4. 信頼性・社会的証明（80/100）✅

**強み（そのまま維持）:**
- 合格率100%（全員合格）— 強力な社会的証明
- 奨学金獲得実績（12名全員合格、学費全額免除）
- 高校生の声3名 + 保護者の声2名
- 講師の実体験（石垣島出身、偏差値7,077位の高校から台湾留学成功）
- 「無理な勧誘なし」の明示

**改善余地:**
- 合格者の声に**写真**がない → 信頼感が下がる
- 「※これらはすべて実在する実績です」の注釈 → 逆に疑念を生む可能性
- 講師紹介に顔写真はあるが、メディア掲載等の第三者証明がない

### 5. ページ構造（45/100 → Clarity実測で裏付け済み）🔴

**問題**: ページが長すぎる — **Clarityデータで実証**

**Clarity実測データ:**
- **80%の訪問者がページの5%（ファーストビュー）で離脱**
- 平均スクロール深度: わずか13%
- 17セクション中、セクション3以降を見るのは全体の15%のみ
- 8大特典（セクション12）、講師紹介（セクション11）にはほぼ誰も到達していない
- 890ページビュー中、合計49タップしか発生していない

**診断:**
- 17セクション、スクロール数推定10回以上
- 「無料説明会の参加登録」という低ハードルなオファーに対してLPが長すぎる
- Clarityで証明: ページ下部のコンテンツはほぼ見られていない

**原則**: オファーのハードルとLPの長さは比例すべき
- 無料説明会 → 短いLP（3-5スクロール）で十分
- 有料商品（¥1,240,000） → 長いLPが必要

**改善案:**
- ファーストビュー、実績、CTA、FAQ の4セクションに凝縮
- 詳細情報は説明会自体で伝える（LPの役割は「登録させる」だけ）
- または: アコーディオン化して必要な人だけ詳細を見れるようにする
