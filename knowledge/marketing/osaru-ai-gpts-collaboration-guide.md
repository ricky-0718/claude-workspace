## 概要

おさるAI GPTs（全56種）はChatGPT上で動作するおさるマーケ講座公式のAIアシスタント群。
Claude Codeから **Playwright MCP経由** でChatGPTの画面を自動操作し、GPTsと対話→成果物を取得→ローカルに保存する。

### なぜClaude Code経由で使うのか

| 直接ChatGPTで使う | Claude Code経由で使う |
|---|---|
| 手動で情報を入力 | knowledge/から自動で事業データを収集・構造化して送信 |
| 1回の出力で完了しがち | 批判的レビュー→改善要求を自動で繰り返す |
| ChatGPT内で完結 | 成果物をローカルに保存・git管理・HTML化まで一貫 |

## 対話ワークフロー（5ステップ）

### Step 1: GPT選定
ユーザーの指示からURLまたはGPT名を特定。GPTカテゴリ一覧（後述）から選ぶ。

### Step 2: 事業データ収集
GPTに送る前に、必ず knowledge/ から関連情報を収集する。
- business/会社全事業データ総覧.md — 会社概要・実績
- marketing/パーソナルナレッジ.txt — 代表プロフィール・ペルソナ
- sales/ — 面談データ・セールス分析
- marketing/osaru-methodology.md — おさる式理論
- その他、タスクに応じた専門ナレッジ

### Step 3: 構造化入力を送信
GPTが求めるフォーマット（番号付き質問等）に合わせて、収集した情報を整理して送信。
- 具体的な数字を入れる（「合格率100%」「12名全員合格」等）
- 断り文句は実データから（過去の面談分析レポートから抽出）
- ターゲットの悩みは生の声で（FAQ・面談履歴から引用）

### Step 4: 批判的レビュー＆ブラッシュアップ（最重要）
GPTの第1版出力をそのまま使わない。以下の観点でレビューする：

| チェック観点 | 例 |
|---|---|
| 質問が決めつけ型になっていないか | 「ありますよね」→ オープンクエスチョンに |
| 具体エピソードが薄くないか | 「大失敗した」→ 「8単位しか取れず絶望」に |
| 保護者/意思決定者への配慮が足りるか | スライド15だけ → 3箇所以上に増設 |
| knowledgeの情報が反映されているか | SPIN型スクリプトの構造と一致するか |
| 温度感・口調がリッキーさんに合っているか | パーソナルナレッジの口調と照合 |

改善要求は具体的に送る：
- 何が問題か（スライドX〜Yの質問が決めつけ型）
- なぜ問題か（相手の本音が引き出せない）
- どう改善するか（オープンクエスチョンに変更。例：「〇〇」）

### Step 5: 成果物の保存・活用
- テキスト版: knowledge/ 配下にMarkdownで保存
- HTMLプレゼン版: 同ディレクトリにHTMLで保存
- Canva移植: 必要に応じてCanva MCPで作成

## Playwright MCP操作の注意点

### ChatGPTの入力方法
テキスト入力は prompt-textarea 要素に対して以下の手順で行う：
1. textarea.focus() でフォーカス
2. DOM操作で段落要素としてテキストをセット
3. input イベントを発火して ChatGPT のリアクティブシステムに反映
4. send-button の data-testid で送信ボタンをクリック

### 応答待ち
- 「思考中」「ストリーミングの停止」のテキストが消えるまで待つ
- 長い応答は3〜5分かかることがある（特に「思考の拡張」ON時）
- 応答が途切れたら「続けてください。スライドXから再開し〜」で続行を依頼

### 応答取得
- data-message-author-role="assistant" で全アシスタントメッセージを取得
- .markdown クラスの要素から innerText を取得
- 複数メッセージに分かれることがある → 全メッセージを結合

### テーブルの取得
- innerText はテーブルをタブ区切りで返す（正しく読める）
- 非常に長い応答はDOMが分割されることがある → 複数メッセージとして取得

## Tips

1. GPTの新バージョンが出ていたら「新しいチャット」で最新版を使う
2. 「思考の拡張」ONの場合: 応答が長くなるが品質も上がる。3〜5分待つ
3. 応答が途切れたら「スライドXから続けて」と具体的に指示
4. ChatGPT Plusの制限: o3モデルは回数制限あり
5. 全GPTsリンク集: osaru-funnel-templates.md の「3. おさるAI（GPTs）一覧」参照
