## 2. サムネイルコピーテンプレート

### 使用パターン一覧

#### Pattern 1: 短期的快楽型（希望・成功イメージ）
**使い方**: 実績の華やかさを強調したい場合、転機を経験した生徒向け

- 「中国語ゼロ→【大学名】合格」
- 「🎓国立大合格の秘訣」
- 「偏差値40→台湾TOP合格」
- 「【生徒名】の逆転劇」
- 「親に反対された留学」

**効果**: 劇的な成長ストーリーに見え、「自分もできるかも」という希望を喚起

---

#### Pattern 2: 逆説・衝撃型（常識を破る）
**使い方**: 留学に対する先入観を打ち破りたい場合、新しい視点を提示する時

- 「偏差値不問で国立に受かる裏道」
- 「日本の大学より安い？」
- 「親が反対した理由」
- 「高卒で台湾TOP合格」
- 「浪人しなくて正解」

**効果**: 視聴者の「え、本当？」という疑問心をくすぐり、クリック率アップ

---

#### Pattern 3: ネガティブ訴求型（恐れ・損失回避）
**使い方**: 決断遅延層へのアプローチ、後悔を避けたい層向け

- 「日本の大学だけだと損する話」
- 「進学前に知るべき現実」
- 「後悔しない大学選び」
- 「国内進学の落とし穴」
- 「中国語スキルなしは危険」

**効果**: 視聴者の「今決めないと失敗する」という危機感を生み、行動喚起へ

---

### テンプレート選択フロー

1. **生徒の成功ストーリーが映える** → **Pattern 1** を選択
2. **留学という選択肢を新しく見せたい** → **Pattern 2** を選択
3. **決断が遅れている見込み客へ動かしたい** → **Pattern 3** を選択
4. **複数の動画がある場合** → パターンをローテーション（同じ訴求ばかりだと飽きられる）

## 3. タイトルテンプレート

### テンプレート5選（YouTube最適化版）

#### Template 1: 数字＋感情（短期的快楽）
**形式**: 「【数字】で【結果】｜【詳細】」
**特徴**: 視覚的インパクト大、スクロール時の目を引く

- 「【偏差値○○】から3ヶ月で国立合格｜台湾留学インタビュー」
- 「5人の合格者が語る｜台湾留学の現実と成功の秘訣」
- 「年間100名以上が選ぶ理由｜台湾留学の5つのメリット」

---

#### Template 2: 疑問形＋解答（クリック誘導型）
**形式**: 「【疑問】？【答え】」
**特徴**: 視聴者の「知りたい欲」に訴える

- 「留学って実際いくらかかる？【合格者が費用を公開】」
- 「本当に中国語0から合格できる？【現役合格者が証明】」
- 「台湾の方が安い？【日本の大学との費用比較】」

---

#### Template 3: 逆説・衝撃型（サムネイル連動）
**形式**: 「【常識を破る言葉】｜【詳細】」
**特徴**: Pattern 2 のサムネイルと連動、クリック率最大化

- 「日本より安く国立に合格｜知られていない台湾留学の真実」
- 「偏差値より大事なもの｜台湾TOP3大学の合格基準」
- 「浪人より留学｜高卒で合格した理由」

---

#### Template 4: ストーリー型＋転機（感情的共感）
**形式**: 「【キャラ】の【状況】→【結果】」
**特徴**: 長めだがサムネイルで顔出しする場合の定番

- 「【生徒名】さんの留学記｜親反対から3年で国立大卒業へ」
- 「中国語初心者のリアル｜台湾留学で人生が変わった話」
- 「不安から希望へ｜受験生親子が語る台湾留学の本音」

---

#### Template 5: 実用型＋スキル（情報検索対策）
**形式**: 「【検索キーワード】【実用情報】」
**特徴**: SEO・関連動画への表示対策、情報系検索ユーザー向け

- 「TOCFL対策講座｜初心者向け勉強法と3ヶ月で取得した秘訣」
- 「台湾大学入試ガイド｜出願から合格まで完全解説」
- 「台湾の生活費は？｜1年の費用内訳と節約テク」

---

### タイトル作成時の5つのチェックリスト

- [ ] **60字以下か？** → YouTubeのタイトルは60字前後で切れるため、大事な情報は冒頭に
- [ ] **数字が入っているか？** → 数字があるだけでクリック率が上がる（統計的実証あり）
- [ ] **サムネイルと一貫しているか？** → タイトル＋サムネイルで「続きが気になる」状態に
- [ ] **感情的なフックはあるか？** → 疑問形、驚き、同情、希望など、何らかの感情を揺さぶるか
- [ ] **ターゲット層に響くか？** → 高卒生向けか、親向けか、すでに興味ある人か、段階で文体を分け

## 4. 編集マニュアル

### 説明欄作成ステップ

1. **テンプレートをコピー** → 上記「概要欄テンプレート」全文を使用
2. **【】内の値を埋める**
   - LINE登録URL: ファネルのLINE登録ページへ
   - 生徒名: 動画に出演している生徒の氏名
   - 進学先: 実際の大学・学部名
   - タイムスタンプ: 編集完了後に動画を見直して正確な時間を記入
   - SNS URL: 全媒体を確認して最新URL記載
3. **ハッシュタグを追加** → 「#台湾留学」「#中国語」など最低5個、最大15個
4. **再確認** → リンク切れ、タイムスタンプの誤り、LINEの重複記載がないか

### サムネイルコピー選択フロー

| 生徒の特徴 | 推奨パターン | 理由 |
|---|---|---|
| 成績が飛躍的に上がった / 偏差値が低かった | Pattern 1 | 成功の快感が最大化 |
| 日本の大学で迷っている / 新しい選択肢を提示 | Pattern 2 | 「台湾という選択肢」を新鮮に見せる |
| 親の反対を乗り越えた / 決断に迷いがあった | Pattern 3 | 後悔回避の心理に訴える |
| 複数の動画がある場合 | ローテーション | 飽きられ防止・A/Bテスト |

### タイトル決定フロー

1. **Template をテンプレート5選から1つ選ぶ**
2. **サムネイルの「言葉」とタイトルを連動させる** → 一貫性がクリック率を高める
3. **チェックリスト5項目を回す** → 特に「数字」「感情」「60字以下」を重視
4. **複数案を出して A/B テスト対応可能に** → YouTubeアナリティクスで「インプレッション」「クリック率」を測定

## 5. コンテンツ最適化チェックリスト

- [ ] 説明欄の1行目（YouTube検索結果に表示される部分）が引き込めているか

- [ ] タイムスタンプが正確か（編集者に確認を取ったか）
- [ ] リンク（LINE、SNS、ブログ）は全て有効か（クリックして確認）
- [ ] ハッシュタグが5〜15個の適正範囲か
- [ ] タイトル + サムネイル + 説明欄の「トリプルメッセージ」が一貫しているか
