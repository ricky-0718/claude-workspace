# 台湾 学生居留ビザ申請ガイド

> ソース: BANDトーク実際のやり取り + 台北駐日経済文化代表処公式ページ
> 公式URL: https://www.roc-taiwan.org/jp_ja/post/25157.html#student
> 注意: 毎年、代表処より事前告知なく規定が変更されることがある。最新情報は必ず公式ページで確認

## ビザ申請と大学出願の財力証明の違い（重要）

| 項目 | 大学出願時 | ビザ申請時 |
|---|---|---|
| **必要書類** | 銀行発行の「残高証明書」 | 銀行発行の「取引明細書・入出金明細書」 |
| **発行条件** | 発行日が3ヶ月以内 | 申請月から遡って**6ヶ月分** |
| **金額要件** | 大学ごとに異なる（USD3,000〜5,000目安） | **50万円以上を6ヶ月間全期間で維持** |
| **言語** | 英語または中国語 | 日本語でOK（日本円表記） |
| **残高推移の証明** | 不要（発行時点の残高でOK） | **必要**（6ヶ月間ずっと50万円以上） |

## ビザ申請時の財力証明 — 詳細条件

### 提出書類
- 銀行発行の取引明細書・入出金明細書

### 条件
- **普通預金口座・定期預金口座** どちらでも使用可能
- **貯蓄預金口座は不可**の可能性が高い（公式案内に記載なし）
- 申請月からさかのぼって**6ヶ月分**の取引明細が必要
- 6ヶ月間**すべての期間において50万円以上の残高が維持**されていること

### 本人名義で条件を満たさない場合
- **三親等以内の親族名義**の取引明細・入出金明細を提出可能
- その場合、親族関係が確認できる公的書類（**戸籍謄本**など）をあわせて提出

### 準備スケジュールの逆算例
- ビザ申請予定: 8月下旬
- → 2月下旬から50万円以上の残高をキープ開始する必要あり
- → 2月下旬〜8月下旬 = 6ヶ月間

### よくある質問（BANDトークから抽出）

**Q: 日本語発行、日本円(50万円以上)での準備で問題ないか？**
A: はい、ビザ申請時の財力証明は日本語・日本円で問題ない。（大学出願時は英語が必要）

**Q: ビザ申請時と大学出願時の財力証明は別物か？**
A: はい、別物。大学出願時は「残高証明書（発行3ヶ月以内）」、ビザ申請時は「6ヶ月分の取引明細（全期間50万円以上維持）」

**Q: 貯蓄預金口座は使えるか？**
A: 公式案内には普通預金・定期預金のみ記載。貯蓄預金口座は不可の可能性が高い

**Q: 推薦書はビザ申請にも必要か？**
A: ビザ申請の際にもう一通必要になる場合がある。その際はスタッフから再案内する

## ビザ申請の全体像

### 申請時期
- 大学合格後、渡航前（8月頃が多い）
- 101センターのスケジュール: 7〜8月にビザ申請

### 申請場所
- **台北駐日経済文化代表処**（東京）
- または各地の弁事処（大阪、横浜、那覇、札幌等）
- 申請場所によっては規定が緩いケースもあるが、全員最も厳しい条件で準備すべき

### 必要書類（学生居留ビザ）
1. パスポート（残存有効期間6ヶ月以上）
2. ビザ申請書
3. 証明写真
4. 入学許可書（大学発行）
5. 財力証明（上記の取引明細 — 6ヶ月分・50万円以上維持）
6. 卒業証明書（英語版 — 大学出願時と同じもの）
7. 成績証明書（英語版 — 大学出願時と同じもの）
8. 入出金明細書（日本語でOK）

### 注意事項
- 合格後は正規留学ビザ申請のために**英語版の書類**が必要
- 大学出願時に日本語版で一時受理された場合でも、ビザ申請時には英語版を用意すること
- 毎年規定が予告なく変更されるため、申請時期が近づいたらスタッフが最新情報をリマインドする

## スタッフの対応フロー

1. **1月頃**: 全体グループでビザ用財力証明のリマインド（りこ担当）
2. **2月〜**: 学生・保護者が残高50万円以上のキープ開始
3. **合格後**: 英語版書類の準備確認
4. **7〜8月**: ビザ申請サポート
5. **渡航前**: 最終確認
