## 1. 翔音さんの目標

| 目標 | 台湾で実現できるか |

## 2. なぜ師範大学なのか

国立台湾師範大学（NTNU）は台湾の教育分野でNo.1の国立大学です。

| 項目 | 内容 |
|------|------|
| 所在地 | 台北市中心部（交通至便） |
| 全校生徒数 | 16,262人 |
| 留学生数 | 1,626人 |
| 日本人留学生数 | 114人 |
| 特徴 | 教育×言語学に強み。台湾政府「2030年バイリンガル国家」政策の中核大学 |

### 翔音さんに師範大学が合う理由

1. **英語×ビジネスの実践型カリキュラム** — 応用外語系では英語とビジネスを組み合わせて学べる
2. **台北で生活できる** — 沖縄からの直行便あり。先輩たちも台北に多数在籍
3. **沖縄出身の先輩が在学中** — 沖縄県の高校出身の先輩が師範大学に合格＆在学中

## 3. おすすめ学科

師範大学にはTOCFL A2で出願できる学科と、B1以上が必要な学科があります。
翔音さんの経歴・目標に合う学科をピックアップしました。

※以下のTOCFL要件は、NTNU公式入学申請システム（bds.oia.ntnu.edu.tw）2026年度データに基づきます。


### 第1志望候補：美術學系 ★ A2で出願可能

| 項目 | 内容 |
|------|------|
| 中国語要件 | **CEFR A2以上（TOCFL A2）** |
| 英語要件 | — |
| 学費/学期 | NT$33,700（約15万円） |
| 所属学院 | 藝術學院 |
| 卒業要件 | 卒業までにTOCFL B1＋英語CEFR B1の取得が必要 |

**デザイン科出身の翔音さんの経験が最も活きる学科。TOCFL A2（3ヶ月で取得可能）で出願できるため、準備のハードルが低い。芸大不合格からの再出発として最適。**

#### 何を学ぶか（4年間のカリキュラム）

| 分野 | 内容 |
|------|------|
| 理論・知識 | 美術史・美学・視覚文化・芸術批評・キュレーション・ギャラリー実習 |
| 書画・水墨 | 水墨画・書道・篆刻・膠彩画 |
| 平面絵画 | 油画・素描・版画・ミクストメディア |
| 立体造形 | 彫刻・陶芸・漆画 |
| 映像芸術 | メディア＆テクノロジー・マルチメディア制作 |

#### 卒業後の進路
- 美術教師（中学・高校）
- アートアドミニストレーター（美術館・ギャラリー運営）
- 美術研究者・美術評論家
- アーティスト・スタジオ経営

> ※出願には**作品集20点以上**（3年以内の作品）の提出が必要です。デザイン科で制作した作品が活用できます。

### 同時出願で合格率を最大化

オールインワンプランでは5校まで同時に出願できます。
A2学科（美術・華語文教學）＋B1学科（企業管理・東亞）を組み合わせて出願し、合格した中から選ぶのが最も現実的です。
