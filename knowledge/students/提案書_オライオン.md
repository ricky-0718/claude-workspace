## 3. おすすめ大学（3校）

### 第1志望：東海大学 — 国際学院 IDP（学際英語学士プログラム）

| 項目 | 内容 |
|------|------|
| 正式名称 | Interdisciplinary Degree Program (IDP) / 國際學院不分系英語學士班 |
| 所在地 | 台中市（台湾中部。台北から新幹線で約50分） |
| プログラム形式 | **1+3モデル**: 1年目は専攻を決めず教養＋中国語。2年目から17学科の中から選択して専攻 |
| 授業言語 | **全英語** |
| 学べること | 1年目に幅広い科目を履修しながら興味を探索。心理学・哲学・社会科学の科目あり |
| 英語スコア | 全英語の学校に在学していた証明書があれば**免除の可能性あり**（要確認） |
| 出願期限 | **第2梯次 〜2026年3月31日** / 第3梯次 4/1〜4/30 / 第4梯次 5/1〜5/27 |
| 学費（1学期） | **NTD 69,020（約31万円）** |
| 年間学費 | 約NTD 138,040（約62万円） |

**なぜ東海大学IDPが最適か：**

1. **入学時に専攻を決めなくていい**
   アメリカの大学と同じように、1年目は色々な科目を取りながら自分に合う分野を見つけられます。2年目以降に社会学・政治学・経済学など17学科から選んで専攻できます。

2. **心理学・哲学の科目が英語で取れる**
   実際に開講されている科目（113学年度実績）：
   - Cross-Cultural Psychology（異文化心理学）
   - Positive Psychology（ポジティブ心理学）
   - Honors: The Problem of Evil（哲学 — 悪の問題）
   - Emotional Management（感情管理）
   ※開講科目は学期ごとに変わります

3. **2年目以降に選べる専攻（17学科）**
   社会学、政治学、経済学、企業管理、国際経営貿易、音楽、環境科学、生命科学 など。人文・社会科学系の選択肢が豊富です。

4. **国際的な環境**
   5大陸350以上の姉妹校との交換留学制度あり。

**出願に必要な書類：**
- パスポート
- 高校の卒業証明書（英語版）
- 高校の成績証明書（英語版）
- 財力証明（USD 4,000以上）
- 自己PR文・学習計画書（英語）
- 英語能力証明（免除申請可能）


### 第2志望：淡江大学 — 外交・国際関係学科 or 全球政治経済学科（全英語）

| 項目 | 外交・国際関係学科 | 全球政治経済学科 |
|------|------|------|
| 正式名称 | Dept. of Diplomacy and International Relations | Dept. of Global Political Economy |
| 所在地 | 新北市淡水区（台北から電車で約40分） | 同左 |
| 授業言語 | **全英語** | **全英語**（公式に「全課程英語授課」と明記） |
| 学べること | 国際関係・政治学・外交・国際法 | 政治学・経済学・社会学・国際政治経済 |
| 特色 | 外交史、国際安全保障、地域研究（北米・EU・東南アジア等） | 1年次に社会学が必修。質的・量的研究方法が充実 |
| 英語スコア | TOEFL iBT 47以上 / IELTS 4.0以上 等（免除条件あり） | 同左 |
| 出願期限 | **第2期 2026年4月16日〜5月6日** / 第3期 5/14〜6/4 | 同左 |
| 学費（1学期） | **NTD 56,260（約25万円）** | 同左 |
| 年間学費 | 約NTD 112,520（約50万円） |  |

**淡江大学の特徴：**

1. **「三全」教育**
   - 全寮制（1年目は寮生活）
   - 全英語授課
   - 3年次に全員1年間の海外留学（留学生は行かない選択も可能）

2. **教員の約50%が外国人、学生の約50%が留学生（45カ国以上）**

3. **全球政治経済学科なら社会学が1年次必修** — 社会科学を基礎からしっかり学べる

4. **全球政治経済学科はNGOインターンシップあり（GSIP）** — 毎年夏に国際NGOで実習


### 第3志望：銘傳大学 — 国際事務・外交プログラム（全英語）

| 項目 | 内容 |
|------|------|

| 英語スコア | TOEFL iBT 47以上 / IELTS 4.0以上 等 |
| 出願期限 | **2026年2月1日〜5月31日** |
| 学費（1学期） | **NTD 46,201（約21万円）** |
| 年間学費 | 約NTD 97,000（約44万円） |

**銘傳大学の特徴：**

1. **台北市の中心部にキャンパス** — 生活の利便性が高い
2. **MSCHE認証** — 米国の大学と同等の教育品質が認証されている。将来的に米国の大学院への進学にも有利

4. **出願締切が5月31日** — 準備に最も余裕がある

## 8. なぜ台湾の全英語課程なのか — マレーシアとの比較

| | 台湾の全英語課程 | マレーシア |
|--|------------|---------|
| 授業言語 | 全英語 | 英語（大学による） |
| 学費 | 年44万〜62万円 | 大学による |
| 沖縄からの距離 | 飛行機で約1.5時間 | 飛行機で約5〜6時間 |
| 中国語も身につく | ○ 生活の中で自然に習得 | × |
| 安全性 | ◎ 治安が良い | ○ |
| 日本語が通じる場面 | 多い（親日国） | 少ない |

台湾の全英語課程なら、英語で専門を学びながら、生活の中で中国語も自然に身につきます。
卒業時に「英語＋中国語＋日本語」のトリリンガルになれるのは、台湾留学の最大の強みです。
