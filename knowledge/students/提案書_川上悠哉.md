## 4. TOEFL受験 — 文化大学に出すために必要なこと

### 必要スコア

文化大学の全英語課程は **TOEFL iBT 57点以上**（またはIELTS 4.0 / TOEIC 550）。
中原大学はCEFR B1以上（TOEIC 550相当。Duolingoも可）。

### 日本でのTOEFL受験日（2026年4月）

| 日付 | 曜日 |
|------|------|
| 4月1日 | 水 |
| 4月4日 | **土** |
| 4月8日 | 水 |
| 4月11日 | **土** |
| 4月15日 | 水 |
| 4月18日 | **土** |
| 4月22日 | 水 |
| 4月25日 | **土** |

※会場は受験地によって異なります。早めの予約を推奨。

### スコアが届くまでの日数

| 方法 | 受験後の日数 |
|------|------------|
| オンライン確認（ETSアカウント） | **4〜8日後** |
| PDF版ダウンロード | オンライン確認の翌日〜2日後 |

### 逆算スケジュール

文化大は第1回（4/15）と第2回（5/4〜6/30）の2回チャンスがあります。

| 日付 | やること |
|------|---------|
| 4月4日（土） | TOEFL iBT 受験（第1回チャンス） |
| 4月8〜12日 | スコア確認（オンライン） |
| → 57点以上なら | 文化大 第1回（4/15締切）にギリギリ間に合う可能性。間に合わなくても第2回に出願 |
| → 57点未満なら | 4月18日（土）or 4月25日（土）に再受験 |
| 5月4日〜 | 文化大 第2回の出願開始 |
| **6月30日** | **文化大 第2回 締切** |

**第2回の締切が6月30日なので、時間的な余裕は十分あります。**
4月にTOEFLを受けて57点に届かなくても、5月にもう1回受けて第2回に出せます。

## 5. 全体スケジュール — 2つのパターン

### パターンA：3月からスタート（おすすめ）

頭金20万円をお支払いいただき、3月中に準備を開始します。

| 時期 | やること |
|------|---------|
| **3月中** | 101センターに申込（頭金20万円）。書類準備を開始 |
| 3月中 | 英語版の成績証明書・卒業証明書を学校に依頼 |
| 3月中 | 推薦書を完成させる（日本語→翻訳→学校印） |
| 3月中 | 自己PR文・学習計画書の作成開始（101センターがサポート） |
| 3月30日 | アメリカの最終確認 |
| **4月上旬** | 中原大学に出願（Duolingo 70で出願。申請費無料） |
| 4月4日（土） | TOEFL iBT 受験 |
| 4月8〜12日 | TOEFLスコア取得（オンライン） |
| **4月15日** | **中原大学 第1回 締切** |
| 5月〜6月 | 文化大学 第2回に出願（TOEFLスコアで出願） |
| **6月30日** | **文化大学 第2回 締切** |
| 6月18日 | 中原大学 合格発表（第1回） |
| 6月〜7月 | 入学手続き |
| 5月〜8月 | 101センターの英語準備コース |
| **9月** | **台湾の大学に入学 → 野球部に入部** |

**メリット**: 書類の準備に1ヶ月以上の余裕がある。自己PR文・学習計画書を丁寧に作り込める。中原大の第1回（4/15）に余裕を持って間に合う。


### パターンB：4月1日からスタート

3月30日のアメリカ確認後に動き出すパターンです。

| 時期 | やること |
|------|---------|
| 3月30日 | アメリカの最終確認 |
| **4月1日** | 101センターに申込。書類準備を一斉に開始 |
| 4月1〜7日 | 英語版証明書を学校に依頼（発行に1〜2週間かかる可能性） |
| 4月1〜7日 | 推薦書の翻訳・押印を急ぎで進める |
| 4月4日（土） | TOEFL iBT 受験（予約が取れれば） |
| **4月15日** | **中原大 第1回 締切 → 書類が間に合わない可能性あり** |
| 4月中旬 | 自己PR文・学習計画書を急ぎで作成 |
| 5月1日〜 | 中原大 第2回に出願（6/15締切） |
| 5月〜6月 | 文化大 第2回に出願（6/30締切） |
| 6月18日 | 中原大 合格発表（第1回） |
| 7月23日〜24日 | 文化大・中原大 合格発表（第2回） |
| 7月〜8月 | 101センターの準備コース（短縮版） |
| **9月** | **台湾の大学に入学** |

**リスク**: 中原大の第1回に間に合わない可能性がある。書類準備が全部同時進行でバタバタする。TOEFL受験の予約が埋まっている可能性。準備コースの期間が短くなる。


### パターン比較

| | パターンA（3月スタート） | パターンB（4月スタート） |
|--|------------------------|------------------------|
| 中原大 第1回（4/15） | ○ 余裕で間に合う | △ 間に合わない可能性 |
| 文化大 第2回（6/30） | ○ 余裕で間に合う | ○ 間に合う |
| 書類の質 | 丁寧に作れる | 急ぎで粗くなる |
| TOEFL予約 | 早めに確保できる | 直前で埋まっているリスク |
| 準備コース | 5月〜8月（4ヶ月） | 7月〜8月（2ヶ月） |
| 費用 | 頭金20万円を3月中に | 4月以降に一括 |

## 8. 101センターのサポート内容

149万円に以下がすべて含まれています。追加料金は一切ありません。

| サポート | 内容 |
|---------|------|
| 出願 | 5校まで同時出願。書類作成・翻訳・提出をすべてサポート |
| 語学 | 英語準備コース（全英語課程向け）or 中国語コース |
| 現地 | 台湾に駐在スタッフ常駐。ビザ・住居・銀行口座の手続き |
| 野球 | 大学野球部の監督への事前連絡・プレイ動画の提出サポート |
| 継続 | 入学前〜卒業後の就職まで5年間の継続サポート |
| 面談 | 毎月の個別面談で学習・生活をフォロー |
| 転学 | 万が一合わない場合の学科変更・大学変更もサポート |
