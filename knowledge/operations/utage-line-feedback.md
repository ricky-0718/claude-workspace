# UTAGE LINE返信 フィードバック記録

ユーザーからの返信案フィードバックを蓄積し、次回以降の返信品質を向上させる。

## 基本ルール
- ビジネス調で統一（「ございます」「いたします」等の敬語を基本）
- 語尾に「ね」を使わない
- 冒頭に `【返信案 M/DD】` タグを付与

## 学んだパターン

### コンテキスト把握が最重要
- 返信前に必ず相手の属性を確認する（参加歴、学年、流入経路、過去の面談有無）
- 高2でサマーキャンプ参加済み → 5期生の営業対象。進路提案に繋げる
- 同じキャンプ参加者の親同士の関係性も意識する（例: RISMANさんと望月さん母）

### 相手の質問の意図を読む
- 「住所はどこですか」→ 文字通りの住所ではなく「説明会はどこで開催？」の可能性が高い
- 説明会広告からの流入者には「オンライン開催です」と案内するのが正しい対応
- 表面的な質問に答えるのではなく、文脈から意図を汲み取る

### トーンの使い分け
- 基本はビジネス調（「ございます」「いたします」等の敬語）で統一する
- ただし、フランクなタメ口でやり取りしている相手には、同じトーンで返す
- 一律に丁寧語にしない。関係性に応じて使い分ける
- ※「語尾に『ね』を使わない」はトーンに関わらず全体共通ルール
