# Freee請求書 作成・送付ガイド

## アクセス
- URL: `https://invoice.secure.freee.co.jp/`
- ログインが必要（認証情報はユーザーが入力）
- 新規作成: `https://invoice.secure.freee.co.jp/reports/invoices/new`
- 見積書一覧: `https://invoice.secure.freee.co.jp/reports/quotations`

## 請求書作成フロー

### 1. 取引先情報
- `取引先` comboboxに会社名を入力 → ドロップダウンから選択
- 敬称: 御中/様/(空白) から選択

### 2. 請求情報
- 請求日: 自動で当日が入る
- 入金方法: 振込（デフォルト）
- 入金期日: 手動入力（例: 月末 `2026-03-31`）
- 件名: 手動入力（例: `翻訳業務`）

### 3. 明細入力
- 各行のフィールド: 摘要、取引日、数量、単位、単価、税率（10%/8%/0%）
- 行追加: 行番号横の `+` ボタン → 「この明細行を複製」「通常行を追加」「テキスト行を追加」
- 割引行: 摘要に「セット割引」、単価にマイナス値、税率0%で非課税にする
- **注意**: `find` で税率セレクタを探す際、行番号のマッピングがずれることがある → `read_page` の `ref_id` でlistitemを指定して正確に特定する

### 4. 税率の注意点
- 割引行の税率を0%にする場合、正しいcomboboxの `ref_id` を確認すること
- 方法: 割引行の親要素（listitem）を `read_page` で読み、その中の combobox を特定

### 5. 備考欄
- テンプレートが自動入力されている場合がある → 全選択して上書き
- `form_input` で直接値をセットすればOK

### 6. 保存
- 「保存」ボタン（青）をクリック → 請求書番号が自動採番される

## メール送付フロー

### 1. 送付画面を開く
- 請求書詳細ページ → 「...」（その他アクション）→ 「メール送付」

### 2. メール設定
- 送付方法: Web共有リンク / PDFファイル添付
- 宛先 (TO): メールアドレスを入力
- CC: カンマ区切りで複数入力可
- 差出人 (From): 表示名を入力（実際の送信元は noreply@freee.co.jp）
- 返信先メールアドレス: `ricky@ryugaku101.com`
- 「CCで返信先メールアドレスにも送信する」チェック → 自分にもCCで届く

### 3. 送信
- 「確認」→ 送信内容プレビュー → 「送信」

### 翻訳業務の単価
- 単価: **8円/字**（2026年2月に合意）
- 対象言語: 繁体字、簡体字、英語
- 品目名: `翻訳業務（繁体字）`、`翻訳業務（簡体字）`、`翻訳業務（英語）`
- 単位: `字`

### 割引の計算方法（合計を端数なしにする場合）
- 3言語の小計を計算（例: 1,838字 × 8円 × 3 = 44,112円）
- 消費税10%を加算（例: 4,411円）
- 目標合計から逆算して割引額を決定（例: 48,523 - 45,000 = 3,523円）
- 割引行は税率0%にする（非課税）
- 備考に案件番号を記載: `H6827041A　P）R7おきなわUT推進`

## 支払通知書の作成

### アクセス
- 一覧: `https://invoice.secure.freee.co.jp/reports/payment_notices?filter_view_id=-1&cancel_status=uncanceled`
- 新規作成: `https://invoice.secure.freee.co.jp/reports/payment_notices/new`

### 作成フロー
1. 取引先をcomboboxに入力 → ドロップダウンから選択
2. 支払通知日（デフォルト当日）、件名を入力
3. 明細行を入力（摘要、数量、単価、税率）
4. **税区分（内税/外税）**: 右下のセレクトボックスで切替。取引先を再選択すると外税にリセットされるので注意
5. 備考欄、支払予定日、支払方法を入力
6. 保存 → PDFダウンロード（「...」→「PDFダウンロード」）

### 注意点
- 取引先を再選択すると税区分が「外税」にリセットされる → 必ず最後に確認
- PDFファイル名: `{取引先}_{敬称}_{件名}_支払通知書_{番号}.pdf`
