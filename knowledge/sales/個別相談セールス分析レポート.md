━━━ Case 1：岩男 隼人 ━━━━━━━━━━━━━━━ ✅ 成約 ━━━
  高3（浪人予定）／ボリビア留学10ヶ月経験／本人のみ→母別日

  ○ よかった点
    ・本人面談→母に説明会→母面談当日に成約。「2段階設計」が機能した
    ・ワーホリ・サマーキャンプなど複数選択肢を柔軟に提示できた
    ・先延ばし思考に「1年間モチベを保つのは難しい」と経験談でリスク提示

  ✕ 問題点
    ・費用の話で「自分で全額は難しい」をスルー。深掘りが甘い
    ・先延ばし発言にリスク提示はしたが、決断を促すクロージングがない
    ・面談の終わりが「お母様も説明会参加してみてください」のみ。受動的

  → 改善アクション
    ・「お母様にも一緒に話を聞いてもらうのが一番早い」と
      保護者面談の日程をその場で確定させるべきだった
    ・費用の話が出た時点で「分割払いもあります。一緒に考えましょう」と先手を打つ


━━━ Case 2：鈴木 志麻 ━━━━━━━━━━━━━━━ ✅ 成約（10日後）━━━
  高2女子（バレエ経験者）／親子同席／娘が体調不良

  ○ よかった点
    ・体調不良に配慮し「優先案内枠を1週間延長」。寄り添い◎
    ・LINEで母の6つの質問に1つ1つ丁寧に回答。信頼を積み上げた
    ・母が自発的に「色々悩み検討して申し込みさせていただきました」と決断

  ✕ 問題点
    ・面談中、娘がほぼ発言していない。「行きたい」を引き出すプロセスなし
    ・クロージングが「1月末までにお返事を」と曖昧。結果2月6日まで引っ張られた
    ・舞台芸術系の学科紹介で娘の意欲を直接確認する質問がない

  → 改善アクション
    ・「娘さんの声を次回15分だけでも聞きたい」と本人との接点を別途設ける
    ・LINEの質問対応の丁寧さを面談後フォローの標準テンプレートに体系化する

  ┌──────────────────────────────────────────────────────┐
  │ 問題 1  面談が「情報提供セミナー」になっている       │
  └──────────────────────────────────────────────────────┘

    今やっていること：
      ヒアリング(5分) → 大学紹介(15分) → サポート説明(10分)
      → 中国語説明(10分) → 「質問は？」 → 「検討します」で終了

    あるべき姿：
      痛みの深掘り(15分) → 理想の未来(10分) → ギャップ認識(5分)
      → 解決策を提案(10分) → クロージング(5分)

    決定的な違い＝「相手に自分の問題を自覚させる時間の長さ」

    例：佐藤瑞季（90分→即キャンセル）
      「Fランしか行けないのか」と悩んで来ているのに、
      すぐ台北科技大学の紹介に入っている。
      本来すべきは「今のままFランに行ったら5年後どうなると思いますか？」
      情報は十分与えた。でも感情を動かせていない。

  └──────────────────────────────────────────────────────┘

      林 直生   → 当日申込希望 → 主治医に反対された
      久保田    → 本人のみ面談 → 両親・カウンセラーに反対された

      ★ 5件中5件が「面談に参加していない意思決定者」が原因

  改善提案  ── 成約率 42% → 70%以上へ
══════════════════════════════════════════════════════════════


  ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
  ┃  改善 1  面談の構造を「SPIN型」に変える ★最重要★     ┃
  ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

    合計45分以内。6フェーズに統一する。

    Phase 1 ─ 状況確認（5分）
      「今の状況を教えてください」
      → 学年・成績・語学・家庭状況を把握

    Phase 2 ─ 問題の深掘り（15分）★ここが核心★
      情報提供より先にここに時間をかける。
      相手が自分で「このままではマズい」と言語化するまで掘り下げる。

        「今のまま日本の大学に行って、4年後どうなると思いますか？」
        「海外留学を諦めたとき、5年後の自分はどう感じると思いますか？」
        「お母様から見て、今の進路のまま進んで、本当に納得できますか？」
        「ぶっちゃけ、一番怖いのは何ですか？」

      ※ リッキーさんが言うのではなく、相手に言わせることが重要

    Phase 3 ─ 理想の未来を描く（5分）
        「費用も語学力も全部クリアできるとしたら、一番行きたい未来は？」
        「トリリンガルで年収1000万の仕事に就いている自分、想像できますか？」

    Phase 4 ─ ギャップの確認（3分）
        「今の状態と理想の未来、このギャップを埋めるには何が必要だと思いますか？」
        → 相手が「環境」「サポート」「語学力」などと答える
        → 「まさにそれを全部用意しています」

    Phase 5 ─ サポート提案（10分）
      ここで初めてオールインワンプランの説明。
      相手が自分で言った課題に対応する形で提示する。

    Phase 6 ─ クロージング（5分）
        「ここまで聞いてみて、やってみたいと思いましたか？」
        → Yes  →「では今日、申込フォームを一緒に記入しましょう」
        → 迷い →「何が引っかかっていますか？」
        → No   →「どの部分が合わなかったか教えてもらえますか？」


  ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
  ┃  改善 2  面談前に「保護者の参加」を必須化する         ┃
  ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

    今すぐルールを変える：個別面談は保護者同席を原則にする。

    予約確認メッセージにこう追加する：

      「個別面談では、留学にかかる全体の費用やサポート体制についても
       詳しくご案内いたします。保護者の方にもぜひ一緒にご参加ください。
       これまで成約いただいた方の9割以上が親子でご参加されています。」

    本人のみで来た場合：
      面談は実施する。ただし終わりに必ず
      「お母様（お父様）と15分だけお話しさせてください。
       費用面のご質問にもお答えできます」と保護者面談を設定する。


  ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
  ┃  改善 3  「即日仮申込」の仕組みを作る                 ┃
  ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

    「面談→検討→熱が冷める→消滅」のパターンを防ぐ。

    面談の最後にこう伝える：

      「今日この場で仮申込（無料）していただければ、
       5期生の枠を7日間確保します。
       正式な契約・入金は7日以内で結構です。
       7日以内にキャンセルされても、一切費用はかかりません。」

    「検討する」を「仮申込して検討する」に変える。

    仮申込の効果：
      ・心理的に「もう一歩踏み出した」状態になる（コミットメント効果）
      ・7日間の具体的な期限ができる
      ・「7日後に確認のご連絡をしますね」とフォローの正当な理由ができる


  ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
  ┃  改善 4  面談後フォローの「3-7-14ルール」             ┃
  ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

    面談後、以下のタイミングで必ず連絡する：

      当日中 → お礼＋面談要約＋次のアクション明示
      3日後  → 「その後ご家族でお話しされましたか？ご質問あればいつでもどうぞ」
      7日後  → 「枠の確保期限が近づいています。状況を教えていただけますか？」
      14日後 → 「まだお気持ちに変わりがなければ、次回の説明会もご案内できます」

    キャンセルLINEが来たときの返し方：

      ✕ ダメな例：
        「承知しました。またの機会に。」

      ○ あるべき返し方：
        「ご連絡ありがとうございます。お気持ちは十分理解できます。
         差し支えなければ、どの部分が一番不安に感じられたかだけでも
         教えていただけますか？
         今後のサービス改善の参考にぜひ聞かせてください。」

    理由を聞くことで：
      (1) 改善のヒントが得られる
      (2) 会話が続くことで再検討のチャンスが生まれる
      (3) 「丁寧に対応してくれた」という印象→将来の紹介につながる

  ┃  改善 5  価格提示を「比較フレーム」で行う             ┃
  ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  ┃  改善 6  申込後の「成功者コミュニティ接触」           ┃

      ・コミュニティへの帰属意識でキャンセルしにくくなる
