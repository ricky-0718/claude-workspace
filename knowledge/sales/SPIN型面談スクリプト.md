┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  Phase 1 ── アイスブレイク＋状況確認 【5分】                    ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  目的：緊張をほぐし、基本情報を把握する
  ゴール：学年・成績・語学力・家庭状況・参加者の関係性を確認

  ──────────────────────────────────────
  ステップ1：挨拶＋名前確認
  ──────────────────────────────────────

    「こんにちは、101センター代表のリッキーです。
     よろしくお願いします。
     〇〇さんでお間違いないでしょうか。」

    ⚠ 注意：保護者の同席を必ず確認する

    → 保護者がいる場合：
      「お母様（お父様）もご一緒ですね。ありがとうございます。
       今日はお二人でお話を聞いていただけるので、
       費用面のことも含めて全部お話しできるので安心です。」

    → 本人のみの場合：
      「今日はお一人で来てくれたんですね。
       費用やサポートの詳しい話もあるので、
       後半だけでもお母様（お父様）に入っていただけると一番いいんですが、
       今日は難しそうですか？」

      → 可能なら後半に保護者合流を依頼
      → 不可能なら面談実施。ただしPhase 6で保護者面談を必ず設定

  ──────────────────────────────────────
  ステップ2：説明会の感想（軽く聞くだけ）
  ──────────────────────────────────────

    「説明会見ていただいてありがとうございます。
     どんな印象でしたか？一言で言うと。」

    ★ ここで長く話させない。感想は1〜2文でOK。
    ★ 目的は相手の「温度感」を測ること。

    → 「すごくいいなと思った」    → 温度高い。Phase 2は短めでOK
    → 「ちょっと気になって」      → 中間。Phase 2でしっかり掘る
    → 「親に言われて」「なんとなく」 → 温度低い。Phase 2が最重要

  ──────────────────────────────────────
  ステップ3：基本情報を手短に確認
  ──────────────────────────────────────

    「現状をちょっと確認させてください。
     今何年生ですか？ 学校の成績はだいたいどのくらいですか？
     英検とか語学系の資格は何か持ってますか？」

    ★ 事前アンケートで把握済みの場合は確認のみ。深掘りしない。
    ★ ここに5分以上使わない。



┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  Phase 2 ── 痛みの深掘り 【15分】 ★★★ 最重要 ★★★          ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  目的：相手が自分で「このままではマズい」と言語化するまで掘り下げる
  ゴール：「変わりたい」「何とかしなきゃ」という言葉を相手から引き出す

  ★ ここが旧スタイルとの決定的な違い ★
  旧：ヒアリング5分 → すぐ大学紹介
  新：痛みの深掘り15分 → 相手が自分で気づく → 解決策を求めてくる

  ──────────────────────────────────────
  質問の流れ（上から順に使う）
  ──────────────────────────────────────

  ① 現状の不満・不安を聞く

    「ぶっちゃけ今の進路について、一番モヤモヤしていることって何ですか？」

    → 相手が答えたら必ず深掘りする：
      「それって具体的にはどういう状況ですか？」
      「いつ頃からそう感じてますか？」

    ┌─────────────────────────────────────────┐
    │ よくある回答パターンと深掘り方                        │
    ├─────────────────────────────────────────┤
    │                                                       │
    │ 「成績が足りなくて行きたい大学に行けない」             │
    │  → 「行けないとしたら、今考えてる選択肢はどこですか？」│
    │  → 「正直、その大学に行って4年後ワクワクしますか？」   │
    │                                                       │
    │ 「留学したいけど親がお金を心配してる」                 │
    │  → 「お金以外の部分では、ご家族は留学に賛成ですか？」  │
    │  → 「もし費用の問題がクリアできたら、行きたいですか？」 │
    │                                                       │
    │ 「何がしたいかわからない」                             │
    │  → 「逆に、絶対やりたくないことはありますか？」        │
    │  → 「5年後こうなりたくないな、っていうのはありますか？」│
    │                                                       │
    │ 「親に言われて来ただけ」（本人の温度が低い）           │
    │  → 「お母様はどうして台湾留学がいいと思ったんですか？」 │
    │    （保護者に質問を切り替える）                        │
    │  → 「〇〇さん自身は、海外に興味はありますか？」        │
    │                                                       │
    │ 「日本の大学でいいかなとも思ってる」                   │
    │  → 「日本の大学に4年間通って、卒業した後の               │
    │     イメージってどんな感じですか？」                   │
    │  → 「周りの先輩で大学行った人、楽しそうですか？」      │
    │                                                       │
    └─────────────────────────────────────────┘

  ② 「このままだとどうなるか」を本人に言わせる

    ★ リッキーさんが言うのではなく、相手に考えさせて言わせる

    「今のまま日本の大学に行ったとして、
     4年後の自分ってどうなってると思いますか？」

    「正直なところ、今の進路のまま進んで、
     後悔しないって胸を張って言えますか？」

    （保護者向け）
    「お母様から見て、このまま普通に日本の大学に行って、
     〇〇さんの可能性を十分に活かせると思いますか？」

    → ここで相手が沈黙したら、待つ。沈黙は考えている証拠。
    → 答えが出てきたら共感する：
      「そうですよね。そう感じるのは自然なことです。」

  ③ 痛みを増幅する（やりすぎ注意。1〜2問だけ）

    「海外留学を今回見送ったとして、
     5年後の自分がそれを知ったら、どう思いますか？」

    「今この瞬間に決断しなかったら、
     次にこういうチャンスが来るのはいつだと思いますか？」

  ──────────────────────────────────────
  Phase 2の成功基準
  ──────────────────────────────────────

  以下のような言葉が相手から出たら、Phase 3に進む：

    ✅ 「やっぱり変わりたい」
    ✅ 「このままじゃダメだと思う」
    ✅ 「なんとかしたいんです」
    ✅ 「行きたいんですけど…」（←この「けど」の後を解消するのがPhase 5）
    ✅ 「娘をなんとかしてやりたい」（保護者）
    ✅ 「今の環境だとワクワクしない」

  ⚠ まだ出ていなければ、Phase 3に進まない。
    追加で聞く：
    「今日わざわざ時間を取って相談に来てくれたわけじゃないですか。
     何か変えたいって気持ちがあるからだと思うんですけど、
     一番変えたいのはどこですか？」



┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  Phase 3 ── 理想の未来を描く 【5分】                            ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  目的：「こうなりたい」というポジティブなビジョンを持たせる
  ゴール：相手の目が輝く瞬間を作る

  ──────────────────────────────────────

    「じゃあ逆に、お金も語学力も全部クリアできるとしたら、
     一番理想の進路ってどんな感じですか？」

    「例えば4年後、中国語と英語と日本語の3つ話せて、
     海外の大学を卒業して、好きな仕事を選べる状態って、
     想像できますか？」

    → 相手が具体的に話し始めたら、広げる：
      「いいですね。それって〇〇さんにすごく合ってると思います。」
      「実際にうちの卒業生で、まさにそういうキャリアの子がいて…」

    → 保護者にも聞く：
      「お母様は、〇〇さんにどんな大人になってほしいですか？」
      「台湾に行った先輩たちの話を聞いて、
       うちの子もこうなれるかもって思いましたか？」

    ┌─────────────────────────────────────────┐
    │ ★ ケース別の理想の描き方                              │
    ├─────────────────────────────────────────┤
    │                                                       │
    │ 成績が低い子                                          │
    │  → 「評定3.2の僕でも台湾の名門大に入れた。            │
    │     日本だとFランって言われる成績でも、                │
    │     台湾だと国立大に受かる。逆転できる。」            │
    │                                                       │
    │ 語学に興味がある子                                    │
    │  → 「トリリンガルって、就活で無敵です。               │
    │     中国語＋英語＋日本語で年収1000万の仕事、          │
    │     普通に狙えます。」                                │
    │                                                       │
    │ 何がしたいかわからない子                              │
    │  → 「わからないのは当たり前。環境が変われば見えてくる。│
    │     日本にいたら絶対出会えない人や経験がある。         │
    │     やりたいことは台湾で見つければいい。」            │
    │                                                       │
    │ 保護者が主導の場合                                    │
    │  → 「今の日本の大学に行っても、就職で差がつかない。    │
    │     台湾留学は"投資"です。4年後の就職力が違う。」     │
    │                                                       │
    └─────────────────────────────────────────┘



┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  Phase 4 ── ギャップの確認 【3分】                              ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  目的：「痛み」と「理想」のギャップを本人に自覚させる
  ゴール：「そのギャップを埋めるには何が必要か」を相手に言わせる

  ──────────────────────────────────────

    「さっき話してくれた今の状況と、理想の未来。
     このギャップを埋めるには、何が必要だと思いますか？」

    → 相手が答える（典型的な回答）：
      「環境を変えること」
      「語学力」
      「サポートしてくれる人」
      「お金」
      「勇気」

    → すかさず：
      「そうですよね。まさにそれを全部用意しているのが、
       うちのオールインワンプランなんです。
       ちょっと具体的に説明させてください。」

    ★ この流れが重要：
      相手が自分で「必要なもの」を言う → それに応える形でサービス紹介
      ＝ 押し売りではなく「求められて提供する」構造になる



┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  Phase 5 ── サポート提案 【10分】                               ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  目的：相手が言った課題に対応する形でサービスを説明する
  ゴール：「これなら自分の問題が解決する」と思わせる

  ──────────────────────────────────────
  ★ 鉄則：全部説明するな。相手の課題に関連する部分だけ説明する
  ──────────────────────────────────────

    相手が言った課題         → 説明すべきポイント
    ──────────────────────────────────────
    「語学力が不安」         → 最短3ヶ月でA2取得の実績、アーカイブ学習
    「一人で海外は怖い」     → 現地駐在スタッフ、サマーキャンプ、5年サポート
    「成績が低い」           → 評定3.2で名門大合格の実例、5学科出願戦略
    「お金が心配」           → 日本vs台湾の比較表（後述）、奨学金サポート
    「何を学べばいいか」     → 毎月の個別面談で一緒に決める、転学サポート
    「親が反対しそう」       → 保護者グループでの情報共有、保護者面談

  ──────────────────────────────────────
  価格提示は「比較フレーム」で行う
  ──────────────────────────────────────

    ★ 149万円を単体で見せない。必ず比較で見せる。

    「ちなみに、日本の私立大学に4年間通うと、
     学費と生活費で合計1,000万〜1,500万かかるのはご存知ですか？

     台湾留学の場合、サポート費149万、学費4年間200万、
     生活費4年間240万で、合計約590万。
     つまり、日本より500万〜900万安い。

     しかも奨学金で最大450万減額の可能性もある。

     この149万は"留学費用"ではなくて、
     "人生を変える投資"として見てもらえると
     ちょっと見え方が変わると思うんですけど、どうですか？」

  ──────────────────────────────────────
  説明は10分以内に収める
  ──────────────────────────────────────

    ⚠ 旧スタイルの失敗パターン：
      大学紹介15分 → サポート説明10分 → 中国語説明10分 → 質疑応答20分
      = 55分以上。相手は疲れて「検討します」で帰る。

    ✅ 新スタイル：
      相手の課題に合わせた説明（5分）→ 価格比較（3分）→ 質疑（2分）
      = 10分。相手はまだエネルギーがある状態でクロージングに入れる。

┃  Phase 6 ── クロージング 【7分】 ★★★ ここで決まる ★★★     ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  目的：「やる/やらない」の意思確認をし、次のアクションを確定する
  ゴール：仮申込 or 保護者面談の日程確定（どちらかは必ず取る）

  ──────────────────────────────────────
  ステップ1：意思確認（直球で聞く）
  ──────────────────────────────────────

    「ここまで聞いてみて、正直どうですか？
     やってみたいって気持ちはありますか？」

    ┌─────────────────────────────────────────┐
    │ 回答別フローチャート                                  │
    ├─────────────────────────────────────────┤
    │                                                       │
    │ 「やりたいです」「行きたいです」                       │
    │  ↓                                                    │
    │  → ステップ2A：即申込へ                               │
    │                                                       │
    │ 「いいなとは思うんですけど…」（迷い）                  │
    │  ↓                                                    │
    │  → ステップ2B：障壁の特定へ                           │
    │                                                       │
    │ 「ちょっと考えたい」「検討します」                     │
    │  ↓                                                    │
    │  → ステップ2C：仮申込の提案へ                         │
    │                                                       │
    │ 「親と相談したい」                                    │
    │  ↓                                                    │
    │  → ステップ2D：保護者面談の設定へ                     │
    │                                                       │
    │ 「ちょっと違うかな」（No）                             │
    │  ↓                                                    │
    │  → ステップ2E：理由の確認へ                           │
    │                                                       │
    └─────────────────────────────────────────┘

  ──────────────────────────────────────
  ステップ2A：即申込（相手がYesの場合）
  ──────────────────────────────────────

    「嬉しいです。では今日、一緒に申込フォームを記入しましょう。
     今からLINEに送りますね。」

    → フォーム送信 → 記入を確認 → 契約書の説明
    → 分割払いの提案：
      「お支払いは一括が基本ですが、
       例えば頭金50万で、残りは4月末までとかも可能です。
       ご家庭の状況に合わせて一番いい形にしましょう。」

  ──────────────────────────────────────
  ステップ2B：障壁の特定（迷っている場合）
  ──────────────────────────────────────

    ★ 「何が引っかかっていますか？」と具体的に聞く

    「気持ちはあるけど迷ってる、その気持ちはすごくわかります。
     一つだけ聞いていいですか？
     今一番引っかかっているのは何ですか？」

    ┌─────────────────────────────────────────┐
    │ 障壁別の返し方                                        │
    ├─────────────────────────────────────────┤
    │                                                       │
    │ 「お金」                                              │
    │  → 「分割もカード払いも対応しています。                │
    │     例えば頭金20万で始めることもできます。             │
    │     お金の問題で夢を諦めるのはもったいないので、       │
    │     一緒に方法を考えましょう。」                       │
    │                                                       │
    │ 「自信がない」                                        │
    │  → 「最初から自信がある子なんて一人もいないです。       │
    │     今活躍してる先輩たちも、最初は同じでした。         │
    │     だからこそ5年間サポートするんです。」              │
    │                                                       │
    │ 「続けられるか不安」                                  │
    │  → 「その不安こそ、一人でやったら確実に起きます。       │
    │     うちは出席率も宿題も管理するし、                   │
    │     サボったら僕から直接連絡します。                   │
    │     だから脱落する子がほとんどいないんです。」          │
    │                                                       │
    │ 「他のエージェントと比較したい」                       │
    │  → 「もちろん比較してください。その上で聞きたいんですが、│
    │     一番大事にしたいポイントは何ですか？               │
    │     費用？サポート期間？合格実績？                     │
    │     それを軸に比べてもらえれば、答えは出ると思います。」│
    │                                                       │
    └─────────────────────────────────────────┘

  ──────────────────────────────────────
  ステップ2C：仮申込の提案（「検討します」の場合）
  ──────────────────────────────────────

    ★ 「検討します」を絶対にそのまま受け入れない ★

    ✕ ダメな返し：
      「わかりました、お待ちしております。」
      （→ これが旧スタイルの最大の失敗。熱が冷めて消滅する）

    ✅ こう返す：
      「もちろんです。大きな決断なので、慎重になるのは当然です。

       一つ提案なんですが、仮申込っていう仕組みがあります。
       今日この場で仮申込（無料）していただければ、
       5期生の枠を7日間確保します。

       正式な契約・入金は7日以内で大丈夫です。
       もし7日以内にやめると決めたら、一切費用はかかりません。

       要は、"枠だけ押さえて、冷静に考える時間を作る"という形です。
       検討するにしても、枠がなくなってからでは遅いので。
       いかがですか？」

    → 仮申込した場合：
      「ありがとうございます。7日後に確認のご連絡をしますね。
       その間、何か質問があればいつでもLINEください。」

    → 仮申込も断られた場合：
      「わかりました。では〇日までにお返事をいただけますか？
       〇日にこちらからご連絡させていただきます。」
      ★ 必ず期限と次のアクションを確定させて終わる

  ──────────────────────────────────────
  ステップ2D：保護者面談の設定（「親と相談」の場合）
  ──────────────────────────────────────

    ★ 「親に聞いてみます」で終わらせない ★

    「そうですよね、大事な決断なのでご家族の了解は大切です。
     一つお願いがあるんですが、
     お母様（お父様）と15分だけお話しさせてもらえませんか？

     費用面とかサポート体制とか、
     保護者の方が気になるポイントって〇〇さんとは違うと思うので、
     直接お話しした方がスムーズです。

     いつ頃ならご都合いいですか？今週中に設定できますか？」

    → 日程が決まったら：
      「〇月〇日〇時ですね。LINEでリマインドしますね。
       お母様向けの資料も事前にお送りしておきます。」

    → 日程が決まらない場合：
      「では、ご家族とお話しされた後にLINEで教えてください。
       〇日までにご連絡いただけますか？
       こちらからも〇日に一度確認させていただきますね。」

  ──────────────────────────────────────
  ステップ2E：理由の確認（Noの場合）
  ──────────────────────────────────────

    「正直に言ってくれてありがとうございます。
     差し支えなければ、どの部分が合わなかったか教えてもらえますか？
     今後の参考にさせてください。」

    → 理由を聞いた上で：
      ・解消できる問題なら再提案
      ・本当に合わないなら無理に引き止めない
      ・ただし「また状況が変わったらいつでも連絡ください」と
        ドアを開けておく

  面談中の禁止事項
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✕ Phase 2の前にサービスの説明に入る
  ✕ 大学の選択肢を3つ以上広げる（迷いの原因になる）
  ✕ 相手の質問に全部答えようとする（時間の浪費）
  ✕ 「何かご質問ありますか？」で面談を終わる
  ✕ 「お待ちしております」「またの機会に」で切る
  ✕ 45分を超える（超えそうなら「残り時間」を宣言して巻く）
  ✕ 保護者面談の日程を確定せずに終わる（本人のみの場合）

  面談中に使うべきキーフレーズ集
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  【共感】
  「そう感じるのは自然なことです」
  「その気持ち、すごくわかります」
  「同じように悩んでた子、めちゃくちゃ多いです」

  【深掘り】
  「それって具体的にはどういうことですか？」
  「一番怖いのは何ですか？」
  「もう少し詳しく聞いてもいいですか？」

  【未来を描く】
  「3つの言語が話せる自分、想像できますか？」
  「4年後、就活で"どこでも選べる"状態って、どうですか？」
  「実際にそうなった先輩がいるんですけど…」

  【クロージング】
  「やってみたいって気持ち、ありますか？」
  「一緒に頑張りましょう」
  「枠がなくなる前に、まず押さえておきましょう」

  【使ってはいけないフレーズ】
  ✕ 「何かご質問ありますか？」（受け身すぎる）
  ✕ 「LINEで聞いてください」（面談で完結させるべき）
  ✕ 「お待ちしております」（待つな。能動的に動け）
  ✕ 「承知しました。またの機会に。」（キャンセルを即受理するな）

  第三者の反対への事前対策（面談の最後に必ず聞く）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  面談の終わりに、必ずこの質問を入れる：

    「ちなみに、この件についてご家族以外で
     相談される方はいらっしゃいますか？
     例えば学校の先生、塾の先生、お医者さんなど。」

  → いる場合：
    「その方向けの資料もご用意できます。
     第三者の方にもちゃんと情報が伝わった方が、
     〇〇さんにとってもスムーズだと思うので。」

  → 事前アンケートで特記事項（ADHD、通院中など）がある場合：
    「通院されているということなので、
     主治医の先生にもご相談されるかもしれませんね。
     医療機関向けの資料もお渡しできるので、
     必要であれば言ってください。」

  面談直後のチェックリスト
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  面談終了後、その日のうちに以下を実施する：

  □ お礼LINE送信（面談要約 + 次のアクション明記）
  □ 約束した資料の送付
  □ カレンダーに次回連絡日をブロック
  □ 面談メモ記録（相手の痛み・理想・障壁・温度感）
  □ 保護者面談の日程が決まっている場合、リマインド設定

  テンプレート（お礼LINE）：

    「〇〇さん、今日はお時間いただきありがとうございました！

     今日のお話をまとめると：
     ・〇〇さんの一番の課題は「△△」
     ・台湾留学で「□□」が実現できる
     ・次のステップは「◇◇」

     〇日までにお返事をいただければと思います。
     何か気になることがあれば、いつでもLINEしてくださいね。」
