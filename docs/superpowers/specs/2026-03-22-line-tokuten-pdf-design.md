- Canva不使用。コードで完全制御

## デザインルール（全フィードバック確定版）

### 1. 中央配置（最重要）
- テキストは全て `text-align: center`
- 左寄せ禁止。右側に不自然な空白を作らない
- `justify-content: center` を全ページで徹底。`flex-start` 禁止

### 2. 均等配置
- ページ内の上下左右の余白を均等に分配する
- 要素間に不自然な大きな空白を作らない
- 要素が少ないページは要素を大きくするか、縦配置にして空白を埋める
- 数字カード等も縦配置で空白を埋める

### 3. カードUI禁止（AIっぽさ排除）
- 角丸カード + ドロップシャドウ禁止
- border-left アクセント禁止
- 数字の表示は罫線区切り + タイポグラフィ型（Aスタイル × Cスタイル）

### 4. 数字の表現
- 大きなフォントで直接表示（容器なし）
- 上下の罫線（`border-top` / `border-bottom`）で区切る
- 複数の数字は縦に積む（小さい区切り線で分離）
- 色: Primary Blue or Warm Accent

### 5. ページ構造
- 青いボーダーフレーム（`border: 4px solid #2B4F82`）
- 上部: 青いヘッダーバー（`background: #2B4F82`）にロゴ + 社名
- 背景: `#dce6f0`（薄い青）
- フッター: 「台湾留学101センター」中央配置

### 6. セクション見出し（内容ページ）
- 全幅の青いバー（`background: #2B4F82`）に白文字

### 7. テーブル（内容ページ）
- ヘッダー行: `background: #2B4F82` + 白文字
- 偶数行: `background: #f8fafc`
- テキスト中央配置

### 8. 強調テキスト
- キーワードを Primary Blue で色付け + 太字
- 数字を Warm Accent で色付け

### 9. 全テキスト太字
- `body { font-weight: 700; }` — 全テキストを太字に統一
- スマホでの視認性を最優先

### 10. スマホファースト文字サイズ（確定値）
- タイトル: **56px** / 統計数字: **72px**（単位40px）
- セクション見出し: **32px** / ラベル: **22px**
- 本文: **22-24px** / テーブル: **18px**
- 統計ラベル: **20px** / フッター: **14px**
- 1ページあたりの文字量を減らし、ページ数を増やす方向

### 11. テキスト量がページに収まるよう行間を調整
- テキスト量が多いページは `line-height: 1.8` に下げる
- 余裕があるページは `line-height: 2.0-2.2` で広めに
