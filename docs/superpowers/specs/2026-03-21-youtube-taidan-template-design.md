# YouTube対談動画 台本テンプレート — 設計書

> 作成日: 2026-03-21
> ステータス: レビュー済み・修正完了

---

## 概要

おさる式対談動画9ステップを台湾留学事業向けにカスタマイズした12ステップ台本テンプレート。4期生（合格済み生徒）＋保護者の対談を1本の動画内で構成し、LINE登録をゴールとする。

## 背景と目的

### なぜ対談動画か
- おさる式原則: 「購入者の9割がYouTube対談動画を最低1本視聴」— YouTubeなければ売上ゼロ
- 台湾留学101センターには現在、対談動画コンテンツがない
- 4期生で合格者が出始めており、撮影可能なタイミング

### 訴求ポイント（2軸）
1. **中国語プログラム**: ゼロから始めてTOCFL合格→大学入学までの語学力成長
2. **合格サポート**: 志望校選定→書類準備→出願→合格までの伴走体制

### ターゲット視聴者
- **高校生（高1〜高3）**: 進路に悩んでいる、日本の大学受験に不安がある
- **保護者（主に母親）**: 費用・安全面・将来の就職に不安がある意思決定者
- **浪人生・既卒者**: 日本の大学に落ちた、別の道を探している

### ファネル上の位置づけ
```
YouTube対談動画 → LINE登録（特典5個以上） → ステップ配信28通 → オートウェビナー → 個別相談
```

## 設計原則

### おさる式から適用するもの
- 8つの心理トリガー（牽引性、社会的証明、信頼、期待感、限定性 等）
- 悩みは一覧で羅列せず1つずつ深掘り
- 特典最低5個以上
- 概要欄冒頭3行にLINE登録リンク
- サムネイル13〜15文字（短期的快楽 / 逆説・衝撃 / ネガティブ訴求）

### 台湾留学向けカスタマイズ
- 「受講前の収益＆悩み」→「留学前の状況＆悩み」
- 「どの販売導線でマネタイズしたか」→「どの大学にどう合格したか」
- 親パートの追加（意思決定者への訴求）
- 収録形式: Zoom（リモート）/ 対面（台湾現地）どちらにも対応

## 各ステップ詳細

### ステップ1: ダイジェスト（30秒）

**目的**: 最初の5秒で視聴を継続させるフック

**構成**:
- 合格発表の瞬間 or 台湾での生活シーン（映像 or 写真）
- 生徒の一番インパクトのある一言を先出し
  - 例:「中国語ゼロから8ヶ月で国立大学に合格しました」
- 親の一言も先出し
  - 例:「正直、最初は反対でした。でも今は…」
- テロップ: 動画の内容を3行で要約

**適用する型**: セールスライティング「型①問題提起」＋「型⑤実績証明」を30秒に凝縮

---

### ステップ2: リッキーによる生徒紹介（1分）

**目的**: 視聴者に「この人の話を聞く価値がある」と認知させる

**構成**:
- 「今日は台湾留学101センターの〇期生、△△さんに来ていただきました」
- 生徒のスペックを簡潔に（出身地、高校、現在の大学）
- 「中国語ゼロからスタートして、〇〇大学に合格。その過程を詳しく聞いていきます」

**収録形式別の対応**:
- Zoom版:「今、台湾の〇〇から繋いでくれています」（現地感の演出）
- 対面版: 台湾の街並みや大学キャンパスが背景に映る構図

---

### ステップ3: 生徒の自己紹介（1分）

**目的**: 視聴者が感情移入できる人物像を描く

**質問テンプレート**:
- Q1.「簡単に自己紹介をお願いします」
  → 名前、出身、高校、現在通っている大学・学部
- Q2.「今、大学ではどんなことを勉強していますか？」
  → 大学生活の具体的なイメージを視聴者に持たせる

---

### ステップ4: 留学前の状況＆悩み（2分）

**目的**: 視聴者に「自分と同じだ」と思わせる（自分ごと化）

**質問テンプレート**:
- Q3.「台湾留学を知る前は、進路についてどう考えていましたか？」
  → 日本の大学受験への不安、偏差値の壁、やりたいことが見つからない等
- Q4.「台湾留学を最初に知ったとき、正直どう思いましたか？」
  → 「中国語できないし無理」「親に反対された」等のリアルな不安
- Q5.「一番不安だったことは何ですか？」
  → 言語、費用、友達、生活、将来の就職…

**演出ノート**:
- 悩みは1つずつ深掘りする（おさる式原則: 一覧で羅列しない）
- リッキーが「それ、めちゃくちゃ分かります」と共感を入れる
- 視聴者の親が抱く不安と重なるようにリードする

**適用する型**: セールスライティング「型①問題提起」＋「型⑥ストーリー共感」

---

### ステップ5: 中国語プログラムの体験（3分）— 訴求①

**目的**: 「中国語ゼロでも大丈夫」を実体験で証明する

**質問テンプレート**:
- Q6.「101センターの中国語プログラムは、どんな内容でしたか？」
  → 授業の頻度、形式（オンライン/対面）、先生の雰囲気、具体的なスケジュール感
- Q7.「最初の頃と今で、中国語力はどれくらい変わりましたか？」
  → TOCFLのレベル変化（ゼロ→A2→B1等）を具体的な数字で
  → 「最初は挨拶も言えなかったけど、今は〇〇ができる」
- Q8.「中国語を学ぶ中で、一番大変だったことと乗り越え方は？」
  → 挫折エピソード→どう乗り越えたか→成長実感
  → 先生やスタッフのサポートがあった場合は自然に引き出す
- Q8b.「中国語ができるようになって、生活や考え方で変わったことはありますか？」
  → スキルだけでなく感情的・人間的な変化（Before→After）を引き出す
  → 「台湾人の友達ができた」「自信がついた」「世界が広がった」等

**引き出したいキーワード**:
- 「ゼロからでも大丈夫だった」
- 「〇ヶ月でTOCFL △△に合格した」
- 「先生が〇〇してくれた」（サポートの具体性）
- 「日本にいながら学べた」or「台湾で毎日使うから伸びた」

**演出ノート**:
- TOCFL証明書やテスト結果があれば画面に表示（実績証明の型）
- 対面収録なら中国語で一言話してもらう（インパクト大）

---

### ステップ6: 合格までのサポート体験（3分）— 訴求②

**目的**: 「自分一人では絶対できなかった」を語らせる

**質問テンプレート**:
- Q9.「志望校はどうやって決めましたか？」
  → 面談で一緒に決めたプロセス
  → 「最初は〇〇大学を考えていたけど、リッキーさんに△△を勧められて」
  → 中国語プログラム vs 全英語プログラムの選択肢
- Q10.「出願の書類準備はどうでしたか？」
  → 自伝、学習計画書、推薦書、認証手続き等
  → 「自分だけでは絶対に無理だった」ポイントを具体的に
  → スタッフがどこまでサポートしてくれたか
- Q11.「合格を知ったとき、どんな気持ちでしたか？」
  → 感情のピーク。ここで視聴者の期待感を最大化
  → 「スタッフに一番最初に報告した」等のエピソード

**引き出したいキーワード**:
- 「書類は全部スタッフがチェックしてくれた」
- 「自伝の添削を何回もしてもらった」
- 「面談で自分に合った大学を提案してもらえた」
- 「一人だったら絶対にここまで来れなかった」

**演出ノート**:
- 出願大学と合否結果を一覧テロップで表示
- 合格通知のスクショがあれば画面共有（許可制）

---

### ステップ7: 合格の瞬間＆進学先について（2分）

**目的**: 実績証明の完了＋理想未来の提示

**質問テンプレート**:
- Q12.「今通っている（or これから通う）大学はどんなところですか？」
  → キャンパス、授業内容、友達、寮生活、台湾ならではの魅力
- Q13.「留学前の自分に一言かけるとしたら？」
  → Before→Afterの総括を本人の言葉で

**適用する型**: セールスライティング「型⑤実績証明」＋「型③理想未来の提示」

---

### ステップ8: 親の登場＆最初の不安（2分）

**目的**: 意思決定者（保護者）に「自分と同じだ」と思わせる

**導入**: リッキー「ここからは、〇〇さんのお母さん（/お父さん）にもお話を伺います」

**質問テンプレート**:
- Q14.「お子さんが台湾留学をしたいと言ったとき、最初はどう思いましたか？」
  → 「正直、不安だった」「反対した」等のリアルな第一反応
  → 費用、安全面、将来の就職、中国語ゼロ…具体的な不安を1つずつ
- Q15.「一番心配だったことは何ですか？」
  → 深掘り。頻出する親の不安:
    - 費用（149万円）
    - 台湾有事
    - 中国語がゼロで大丈夫か
    - 子どもが途中で帰ってこないか
    - 就職に不利にならないか

**演出ノート**:
- 親が本音で話せる空気をリッキーが作る:「多くの保護者の方が最初は同じように感じるんですよ」
- 悩みは1つずつ掘り下げる（おさる式原則）
- Zoom版: 親は別端末 or 横に座って参加
- 対面版: 生徒の横に座っている自然な構図

**適用する型**: セールスライティング「型⑥ストーリー共感」の親バージョン

**親が参加できない場合のフォールバック**:
- 優先順: (A) 別日にZoomで親だけ収録→編集で差し込む > (B) 生徒が「親は最初こう言っていた」と代弁 > (C) 親からのテキストメッセージを画面表示＋リッキーが代読
- フォールバック(B)(C)の場合、ステップ8-9を合わせて2分に短縮

---

### ステップ9: 決め手＆今の気持ち（2分）

**目的**: 不安→安心への転換。親の口から「大丈夫だった」と言わせる

**質問テンプレート**:
- Q16.「最終的に、台湾留学を応援しようと思った決め手は何でしたか？」
  → 引き出したい回答:
    - 「101センターのサポート体制を聞いて安心した」
    - 「面談でリッキーさんが〇〇と言ってくれて」
    - 「他の保護者の話を聞いて」
    - 「子どもの本気度を見て」
- Q17.「お子さんが合格したとき、どう感じましたか？」
  → 感情のピーク（ステップ7のQ11と対になる）
- Q18.「同じように迷っている保護者の方に、何か伝えたいことはありますか？」
  → 親から親へのメッセージ = 最強の社会的証明
  → 「迷っているなら、まず話を聞いてみてほしい」が理想的な着地

**引き出したいキーワード**:
- 「サポートがしっかりしていたから任せられた」
- 「子どもが見違えるように成長した」
- 「日本の大学では得られない経験」
- 「費用以上の価値がある」

**演出ノート**:
- Q18の回答はCTA（ステップ12）への伏線
- 親の言葉に生徒が横でリアクションする姿も映す

---

### ステップ10: これからの目標（1分）

**目的**: 未来への期待感で視聴者を高揚させる

**質問テンプレート**:
- Q19.「これから台湾でどんなことに挑戦したいですか？」
  → 大学での学び、将来のキャリア、やりたいこと

**演出ノート**: 短く、前向きなトーンで締める

---

### ステップ11: 視聴者へのメッセージ（1分）

**目的**: 見込み客の背中を押す

**質問テンプレート**:
- Q20.「台湾留学を迷っている同世代の人に、一言お願いします」
  → 「迷っているなら一歩踏み出してほしい」
  → 「中国語ゼロでも本当に大丈夫」
  → 「あの時決断した自分に感謝している」

**演出ノート**: カメラ目線（視聴者に直接語りかける構図）を意識

---

### ステップ12: リッキーまとめ＆CTA（1分）

**目的**: LINE登録への誘導

**構成**:

**① 動画の振り返り（15秒）**
「〇〇さん、本当にありがとうございました。中国語ゼロから〇〇大学に合格するまでの話、すごくリアルで参考になったと思います」

**② 視聴者の不安に寄り添う（15秒）**
「今この動画を観ている方の中にも、台湾留学に興味はあるけど不安がある、という方が多いんじゃないかと思います」

**③ CTA: LINE登録＋特典（30秒）**
「概要欄のリンクからLINE登録していただくと、〇〇をプレゼントしています」

特典（5個以上。おさる式原則）:
1. 台湾留学まるわかりガイドブック（PDF）— 既存ガイドブックLP特典を流用（存在確認済み）
2. 費用シミュレーションシート — **要新規作成**
3. 大学選び完全ガイド（〇〇校収録）— **要新規作成**（knowledge/students/university-database.md をベースに）
4. TOCFL対策スタートキット — **要新規作成**
5. 過去の合格実績データ集 — knowledge/students/admission-results-3rd-cohort.md をPDF化

> 注: 特典②③④は動画公開前に制作が必要。制作タスクは実装計画に含める

「無料の個別相談もLINEから受け付けています。〇〇さんのお母さんもおっしゃっていたように、まずは話を聞いてみるだけでも大丈夫です」
← ステップ9のQ18（親の言葉）を伏線回収

**④ エンディング**
「チャンネル登録もよろしくお願いします。それではまた次の動画で」

**演出ノート**:
- LINE登録リンクは概要欄の冒頭3行に配置（おさる式原則）
- 特典一覧をテロップで表示
- 登録QRコードを画面端に常時表示（Zoom/対面共通）

### 1本目の選定基準（優先順）
1. **合格確定済み**であること（出願中は結果待ちで収録不可リスク）
2. **親の参加が確保できる**こと（フォールバックより本人出演が圧倒的に強い）
3. **中国語プログラム経験者**であること（訴求①を語れる）
4. ストーリーの差別化（見込み客の属性に近い人物を優先）

→ 現時点では**上野航**さんが唯一の合格確定者で最優先候補

## 収録前チェックリスト

### 共通（Zoom/対面）
- [ ] **撮影・公開同意書**の取得（生徒＋保護者。未成年の場合は保護者署名必須）
- [ ] 生徒に質問リスト（テーマ概要版）を事前共有 ※全質問は渡さない。「こういうテーマで聞きます」程度に留め、自然な回答を引き出す
- [ ] 親の参加を確認・スケジュール調整
- [ ] 合格通知・TOCFL証明書等の素材を事前収集
- [ ] テロップ用データの準備（出願校一覧、スペック情報）
- [ ] LINE登録URL・QRコード画像の準備
- [ ] 特典コンテンツの存在確認

### Zoom版追加

## 編集ガイドライン（概要）

> 詳細な編集マニュアルは本スペックのスコープ外。以下は最低限の方針。

- **テロップ**: 重要キーワード・数字は必ずテロップ表示。中央揃え（おさる式リール編集原則に準拠）
- **BGM**: 爽やか系（感動系は過剰にならないよう注意）。親パート（ステップ8-9）は少し落ち着いたトーンに切り替え
- **カット**: 余分な間は全カット（おさる式原則）。言い直し・噛みも除去
- **トランジション**: シンプル（フェードまたはストレートカット）。派手なエフェクト不要
- **ブランドロゴ**: 動画冒頭に101センターのロゴ表示。画面右下に小さく常時表示
- **LINE QRコード**: ステップ12のCTAパートで画面中央に大きく表示。それ以外は右下に小さく常時表示

## 概要欄タイムスタンプについて

構成表の時間配分は収録時の目安。編集後の実際の尺は変動するため、**タイムスタンプは編集完了後に実測値で記入する**。概要欄テンプレートのタイムスタンプはプレースホルダーとして扱うこと。
