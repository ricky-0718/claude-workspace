# 台湾留学LP ウタゲ実装設計

## 概要

完成済みの `taiwan-lp-final.html` をウタゲのLP機能に実装する。
ハイブリッドアプローチ（HTML埋め込み + ウタゲネイティブ機能）を採用。

## アプローチ: ハイブリッド

### 構成

```
ウタゲ LP設定
├── Meta Pixel: ウタゲ管理画面で設定済み（追加作業なし）
├── LINEボタンURL: https://utage-system.com/line/open/Dpbj9HjjOAO3
└── ページ本体: HTML埋め込みブロック
     └── taiwan-lp-final.html (改修版)
```

### 選定理由

| 方式 | デザイン再現 | トラッキング | 編集しやすさ |
|------|------------|------------|------------|
| フルHTML埋め込み | ◎ | △ 手動追加 | × HTMLのみ |
| LPビルダー再構築 | △ 制限あり | ◎ ネイティブ | ◎ GUI |
| **ハイブリッド** | **◎** | **◎ ウタゲ設定** | **○ コメント付き** |

## HTML改修内容

### 1. LINEリンク差し替え

- 変更前: `href="#line"` （全9箇所）
- 変更後: `href="https://utage-system.com/line/open/Dpbj9HjjOAO3"`
- 対象: `.cta-inline` ボタン × 7 + `.cta-btn` 固定ボタン × 1 + ページ内リンク

### 2. 変更しやすいポイントにHTMLコメント追加

- 日程・時間
- 講師名・実績数字
- 特典内容
- 金額比較の数字

### 3. フッターリンク更新

- プライバシーポリシー、特商法、会社概要の実URLを設定

### 4. ウタゲ埋め込み用の調整

- `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>` タグの扱いを確認
- ウタゲの埋め込み形式に合わせて構造調整

## ウタゲ側設定

- Meta Pixel: 設定済み（変更不要）
- OGP: ウタゲ管理画面でタイトル・説明文・サムネイルを設定
- LINE URL: HTML内に直接記載

## 作業フロー

1. HTMLを改修（LINE URL差し替え、コメント追加、フッター更新）
2. ウタゲ管理画面でLPページを新規作成
3. HTML埋め込みブロックに改修済みHTMLを貼り付け
4. プレビュー確認（デザイン、LINE遷移、レスポンシブ）
5. Meta Pixelの発火確認（ブラウザの開発者ツールで確認）

## 懸念事項

- ウタゲのHTML埋め込みが `<style>` タグをサポートするか確認が必要
- 固定CTAボタン（`position: fixed`）がウタゲのレイアウトと干渉しないか確認
- レスポンシブデザインがウタゲのiframe/コンテナ内で正しく動作するか確認
