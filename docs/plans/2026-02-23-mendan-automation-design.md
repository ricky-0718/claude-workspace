# /mendan スラッシュコマンド 設計書

**作成日**: 2026-02-23
**ステータス**: 承認済み

## 概要

面談後の分析・フォロー業務を半自動化する Claude Code スラッシュコマンド。
HiNotes の文字起こしを貼り付けるだけで、分析レポート・LINE文面・Asanaタスク・DB更新が一括実行される。

## ワークフロー

```
面談終了 → HiNotesからコピー → Claude Codeで「/mendan」実行
  ↓
① 面談者情報を入力（名前・学年・同席者）
② 文字起こしテキストを貼付
  ↓ Claude Codeが自動実行
③ 分析レポート生成 → mdファイル保存
④ LINE文面4通生成 → mdファイル保存
⑤ Asanaにフォロータスク4件作成
⑥ Googleスプレッドシートに面談データ追記
```

## コンポーネント

### 入力
- `/mendan` で起動
- 面談者名、学年、保護者同席有無を質問
- 文字起こし全文を受け取る

### 分析エンジン（ナレッジ参照）
- `SPIN型面談スクリプト.md` → SPIN各Phaseの実行度を評価
- `面談後フォロー3-7-14ルール.md` → テンプレートを面談内容でカスタマイズ
- `個別相談セールス分析レポート.md` → 過去12件との比較パターン

### 自動抽出項目
| 項目 | 説明 |
|------|------|
| 痛み | 相手が語った現状の不満 |
| 理想 | 相手が描いた将来像 |
| 障壁 | 決断を妨げている要因 |
| 温度感 | 高/中/低 |
| 意思決定者 | 本人/母/父/その他 |
| 面談時間 | 文字起こしのタイムスタンプから推定 |
| SPIN評価 | 各Phaseの実行度（5段階） |

### 出力A: 分析レポート
- 保存先: `ナレッジ格納庫/面談分析/YYYY-MM-DD_名前.md`
- フォーマット:
  - 基本情報（名前・学年・同席者・面談時間）
  - よかった点（3つ）
  - 問題点（3つ）
  - 改善アクション（3つ）
  - SPIN評価スコアカード

### 出力B: LINE文面
- 保存先: `ナレッジ格納庫/LINE下書き/YYYY-MM-DD_名前.md`
- 生成内容:
  - 当日お礼LINE（面談要約入り）
  - 3日後フォローLINE
  - 7日後期限通知LINE
  - 14日後最終確認LINE

### 出力C: Asanaタスク
- プロジェクト: オールインワンプラン契約書（GID: `1209960384497212`）
- 構造: メインタスク1件 ＋ サブタスク4件

**メインタスク:**
| タスク名 | 期日 | 説明 |
|---------|------|------|
| 〇〇さん 面談フォロー | 面談+14日 | 面談分析レポート全文（基本情報・抽出項目・SPIN評価・よかった点・問題点・改善アクション） |

**サブタスク（メインタスクの子タスク）:**
| サブタスク名 | 期日 | 説明 |
|-------------|------|------|
| 【当日】お礼LINE送信 | 面談当日 | LINE文面ファイルの「当日」セクションを送信 |
| 【3日後】進捗確認LINE | 面談+3日 | LINE文面ファイルの「3日後」セクションを送信 |
| 【7日後】期限通知LINE | 面談+7日 | LINE文面ファイルの「7日後」セクションを送信 |
| 【14日後】最終確認LINE | 面談+14日 | LINE文面ファイルの「14日後」セクションを送信 |

### 出力D: Googleスプレッドシート
- スプレッドシートID: `1lq2N5KtvdNaFqZweEaN2n3nAoFuHRrWKUU8TIseZN_M` (既存の広告分析ダッシュボード) or 新規作成
- シート名: 「面談DB」
- 列構成:

| 列 | 内容 |
|----|------|
| A | 面談日 |
| B | 名前 |
| C | 属性（学年・性別） |
| D | 同席者 |
| E | 温度感 |
| F | 障壁 |
| G | 面談時間（分） |
| H | ステータス（検討中/仮申込/成約/失注） |
| I | 次回アクション日 |
| J | 次回アクション内容 |
| K | 成約日（該当時） |
| L | 備考 |

## 技術的な実装方針

### スラッシュコマンド
- `.claude/commands/mendan.md` にプロンプトテンプレートを配置
- ナレッジファイル3つを参照指定
- Asana MCPツールを使用してタスク作成
- ブラウザ操作（Edge MCP）でスプレッドシートに追記
