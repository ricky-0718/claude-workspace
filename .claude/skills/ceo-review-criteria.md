# 経営者視点レビュー 判断基準

## ROI判定基準（数値ベース）

### 施策の投資回収ルール
- **即着手すべき**: コストゼロで効果が出る施策（面談スクリプト改善、LP切替など）
- **1週間以内**: 実装工数1日以内で、月間+10万円以上の効果が見込める施策
- **要検討**: 実装に3日以上かかり、効果が不確実な施策
- **後回し**: 実装に1週間以上かかり、他にコストゼロの未着手施策がある場合

### 事業のボトルネック優先順位（統合提案書より）
1. LP登録率 0.89% → 3%改善で売上3.4倍（年間+数千万円）
2. LINE配信枠 利用率0.58% → 追加コストゼロで改善可能
3. キャッシュフロー構造問題 → 倒産リスク排除（最重要）
4. 成約率42% → 面談スクリプト改善だけで55%到達可能（コストゼロ）
5. コンテンツ資産死蔵 → 年間+900万円の収益化余地
6. TSMC連携未活用 → プレミアムプラン+ブランド強化
7. バックオフィス月128時間 → 39.5時間に削減可能（-70%）

### 判定ルール
施策を評価するとき、以下の順で問う：
1. 上記ボトルネック1〜7のどれに対応するか？→ 対応しないなら優先度は低い
2. 同じボトルネックに対して、もっとコストが低い打ち手はないか？
3. この施策は「仕組み」になるか、「1回限り」か？→ 仕組み化できるものを優先

## おさる式 売上方程式での判定

```
売上 = リスト数 × LTV
LTV = 信頼残高 × オファーの強さ × 商品単価 × 商品数
```

施策がこの方程式のどの変数に効くかを明示すること：
- リスト数を増やす施策か？（LP改善、広告、SNS）
- LTVを上げる施策か？（信頼残高、オファー改善、商品ライン拡充）
- どちらにも効かないなら、それは経営的に意味があるのか？

## 現在のフェーズ判定

おさる式ロードマップにおける現在地：
- フェーズ1 STEP5-6（スクール型 + ファネル強化）→ フェーズ2 STEP7（エバー化）への移行期
- オートウェビナーファネルは構築済みだが、LP登録率がボトルネック
- この段階で「新しいことを始める」より「既存ファネルの穴を塞ぐ」方がROIが高い

## レビュー時の数値チェック項目

| チェック項目 | 判定基準 |
|---|---|
| この施策は月いくらの売上増に繋がるか？ | 具体的な数字で答えられないなら判断保留 |
| 実装にかかる時間は？ | 3日以上なら、コストゼロの未着手施策がないか先に確認 |
| 失敗した場合のダウンサイドは？ | 取り返しがつかないリスクがあるなら慎重に |
| 同じ効果をもっと簡単に達成する方法はないか？ | 常に最小コストの選択肢を探す |
