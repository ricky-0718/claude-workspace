# 顧客視点レビュー 判断基準

> このファイルはロールプレイではなく、実際の個別相談12件（成約5件・失注5件・未確定2件）から抽出したデータに基づく。
> 出典: `knowledge/sales/個別相談セールス分析レポート.md`

## 顧客プロファイル（実データ）

### メインターゲット
- **高校2-3年生**（親子で意思決定）
- **保護者**（特に母親が情報収集、父親が最終決裁のケースが多い）
- **一部社会人**（Fラン大回避、キャリアチェンジ目的）

### 面談に来る人の共通動機
- 「このまま日本の大学でいいのか」という漠然とした不安
- 海外に興味はあるが、費用と難易度が壁
- 比較対象: 他の留学エージェント、日本の大学進学

## 顧客の5大不安（実データから抽出）

### 1. 費用への不安（12件中10件で出現）
- 「149万は高い」が第一反応
- **解決済みのケース**: 日本の大学4年間(1,080万)との比較を出した瞬間に納得（小田ケース）
- **未解決のケース**: 比較フレームなしで価格を出すと「高い」で止まる
- **判定基準**: 費用の話が出る前に「比較フレーム」が提示されているか？

### 2. 保護者の反対（失注5件中5件が該当）
- 林: 主治医に反対された
- 久保田: 両親・カウンセラーに反対された
- 福井: 母が不参加→音信不通
- 稲福: 「母と相談してから」で保留
- 柳生: 「父にも聞かないと」で消滅
- **鉄則**: 面談に参加していない意思決定者が存在する場合、成約率は極端に下がる
- **判定基準**: 「保護者への説明導線」が設計に含まれているか？

### 3. 「本当に大丈夫か」という信頼不安
- 初めて知ったサービスに149万を払うハードル
- **成約に効いた要素**: 実名の合格者の声、保護者の声、TSMC連携の具体性
- **効かなかった要素**: テキストだけの説明、抽象的な「サポートします」
- **判定基準**: 具体的な「人」の証言があるか？数字で裏付けられているか？

### 4. 「自分にもできるか」という能力不安
- 中国語ゼロから大丈夫か？
- 勉強についていけるか？
- **成約ケースで効いた対処**: 「英検2級→トリリンガル」の具体ストーリー、「評定3.5でも合格」の実績
- **判定基準**: 「自分と似た状況の人が成功した」事例が提示されているか？

### 5. 就職・進路への不安
- 台湾の大学を出て日本で就職できるか？
- **効果的な対処**: TSMC連携、トリリンガルの就職優位性の具体数字
- **判定基準**: 「留学後の出口」が明確に描かれているか？

## 離脱パターン（LPデータから）

### ファーストビュー（0〜3秒）
- 80%がここで離脱する（Clarity実測）
- 判定: 「自分向けだ」と3秒で伝わるか？
- 失敗パターン: 抽象的なキャッチコピー、情報が多すぎる、文字が小さい

### スクロール5%地点
- 到達者はわずか20%
- ここまで残った人は比較的読む（5%→15%の離脱は緩やか）
- 判定: ファーストビューの次に「引きつける要素」（比較表、衝撃的数字）があるか？

### 平均滞在28秒
- 「読む」のではなく「見る」行動
- 判定: テキスト量は適切か？ビジュアルで伝わるか？

## 意思決定プロセス（実データ）

### 成約までの平均日数
- 面談内成約: 1件（光岡）
- 7-10日後: 3件（鈴木10日、小田7日、森塚9日）
- 即日+母面談: 1件（岩男）

### 意思決定の構造
```
本人の興味 → 保護者への相談 → 保護者の納得 → 費用の合意 → 申込
              ↑ここで止まるケースが最多
```

### 成約のトリガー（実データ）
1. 保護者が直接説明を聞けた（成約5件中4件が親子同席 or 保護者面談あり）
2. 費用の比較フレームが提示された（日本の大学 vs 台湾留学）
3. 「自分と同じ状況の人」の成功事例を見た
4. 枠の限定性（「あと5名」）

### 失注のトリガー（実データ）
1. 面談に参加していない意思決定者の反対（5/5件）
2. 「検討します」後のフォロー不足（3/5件）
3. 面談時間が長すぎて疲弊→冷却（佐藤90分→即日キャンセル）
4. 第三者（主治医、カウンセラー）の反対への準備不足

## レビュー時の必須チェック（顧客視点）

1. **3秒テスト**: パッと見て「高校生/保護者の自分向け」と分かるか？
2. **費用フレーム**: 149万が「高い」ではなく「お得」に見える比較があるか？
3. **保護者導線**: 保護者が「これなら安心」と思える情報（安全性、費用対効果、就職）があるか？
4. **同類事例**: 「自分と同じ状況の人」の成功体験が具体的に提示されているか？
5. **不安の先回り**: 5大不安（費用・保護者・信頼・能力・就職）のうち、触れていないものはないか？
6. **行動ハードル**: 次のアクション（LINE登録・申込）のハードルは十分低いか？
7. **モバイル体験**: スマホで見たときに読みやすいか？28秒で要点が伝わるか？
