# /review-marketer — マーケティング責任者視点レビュー

直前の作業内容、または指定された施策・成果物を**マーケティング責任者の目線**でレビューしてください。

## 判断基準（必ず読み込んでからレビューすること）

**`.claude/skills/marketer-review-criteria.md` を最初に読み込む**こと。このファイルには：
- ファネル基準値（各ステップの実測値・目標値・業界平均）
- ボトルネック判定ルール
- LP行動データ（Clarity実測: スクロール深度13%、80%がFVで離脱）
- A/Bテスト判定基準
- チャネル別の判定基準

このSkillsファイルの数値と照合してレビューする。「感覚」での判断は禁止。

## レビューの問いかけ（5つ）

1. **ボトルネック判断**: この施策はファネルのどのステップに効くか？そのステップは現在ボトルネックか？（LP登録率0.89%が最優先）
2. **KPI設計**: 成否を測る具体的な数値目標があるか？測定方法は確立されているか？
3. **ターゲット整合性**: 高校2-3年生/保護者/社会人のうち、誰に向けたものか明確か？
4. **テスト可能性**: A/Bテスト設計基準（7日間・100UU以上・変数1つ）を満たせるか？
5. **チャネル最適化**: LINE配信枠（利用率0.58%）の活用は検討したか？
